# JavaScript Lab

Este lab organiza o estudo de JavaScript moderno como base técnica para desenvolvimento fullstack.

## Objetivo

Consolidar JavaScript como linguagem transversal para navegador, Node.js, consumo de APIs, manipulação de dados, modularização, assincronismo e preparação para TypeScript.

## Como este lab está organizado

- `modules/`: módulos de estudo em sequência.
- `docs/`: página navegável e projetos de conclusão de módulo.
- `docs/projects/`: projetos práticos feitos ao final de cada módulo.
- `docs/assets/`: arquivos compartilhados de CSS, JavaScript e imagens para a documentação/projetos.

## Fluxo de estudo recomendado

1. Leia o `README.md` do módulo.
2. Estude cada tópico pela ordem apresentada em `topics/`.
3. Use `lessons.md` para estudar os conceitos do tópico.
4. Responda `assimilation-questions.md` para fixação.
5. Pratique em `exercises/` quando houver código.
6. Marque o progresso no `checklist.md` do módulo.
7. Ao finalizar o módulo, desenvolva o projeto correspondente em `docs/projects/`.
