# JS-06 — Módulos e organização de código

## Objetivo do módulo

Ensinar como dividir código em unidades reutilizáveis e interoperáveis, com prioridade para ECMAScript Modules.

## Pré-requisitos

JS-05.

- `JS-06.06` — ESM no navegador
- `JS-06.07` — ESM no Node.js
- `JS-06.08` — CommonJS e interoperabilidade
- `JS-06.09` — package.json e campo type
