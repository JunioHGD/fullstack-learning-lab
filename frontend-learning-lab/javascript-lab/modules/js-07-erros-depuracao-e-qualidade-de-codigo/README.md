# JS-07 — Erros, depuração e qualidade de código

## Objetivo do módulo

Ensinar a ler falhas, lançar erros, tratar exceções e depurar comportamento com práticas defensivas.

## Pré-requisitos

JS-06.

## Tópicos

- `JS-07.01` — Erros de sintaxe, runtime e lógica
- `JS-07.02` — Error e erros nativos
- `JS-07.03` — throw
- `JS-07.04` — try, catch e finally
- `JS-07.05` — Stack trace e leitura de mensagens de erro
- `JS-07.06` — console.log, console.error, breakpoints e debugger
- `JS-07.07` — Práticas defensivas e validação básica
- `JS-07.08` — Strict mode como apoio à qualidade de código
