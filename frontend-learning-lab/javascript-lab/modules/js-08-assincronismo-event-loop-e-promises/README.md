# JS-08 — Assincronismo, event loop e promises

## Objetivo do módulo

Construir a base mental do assincronismo em JavaScript, conectando event loop, callbacks, promises e async/await.

## Pré-requisitos

JS-07.

## Tópicos

- `JS-08.01` — Modelo de execução e event loop
- `JS-08.02` — Tarefas assíncronas e callbacks
- `JS-08.03` — Timers
- `JS-08.04` — Promises: criação, estados e encadeamento
- `JS-08.05` — Tratamento de erro com promises
- `JS-08.06` — async function
- `JS-08.07` — await
- `JS-08.08` — try/catch com código assíncrono
- `JS-08.09` — Coordenação com Promise.all, Promise.allSettled, Promise.race e Promise.any
