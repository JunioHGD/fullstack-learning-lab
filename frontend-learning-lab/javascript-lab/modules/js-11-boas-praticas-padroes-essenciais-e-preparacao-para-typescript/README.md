# JS-11 — Boas práticas, padrões essenciais e preparação para TypeScript

## Objetivo do módulo

Consolidar hábitos de escrita previsível, modular e legível, preparando a transição gradual para TypeScript.

## Pré-requisitos

JS-10.

## Tópicos

- `JS-11.01` — Escrita explícita e previsível
- `JS-11.02` — Preferência por const e imutabilidade em nível prático
- `JS-11.03` — Uso criterioso de ===
- `JS-11.04` — Evitar coerções obscuras e efeitos colaterais desnecessários
- `JS-11.05` — Funções pequenas e fronteiras claras de módulo
- `JS-11.06` — Dados com formato consistente
- `JS-11.07` — Convenções de nomeação e legibilidade
- `JS-11.08` — JSDoc e type checking gradual em JavaScript
- `JS-11.09` — Relação entre JavaScript bem escrito e migração para TypeScript
