# JS-10 — JavaScript no Node.js e diferenças de runtime

## Objetivo do módulo

Mostrar o que muda quando JavaScript roda fora do navegador, com foco em Node.js como runtime e sem aprofundar backend.

## Pré-requisitos

JS-09.

## Tópicos

- `JS-10.01` — O que é Node.js como runtime JavaScript
- `JS-10.02` — Diferenças centrais entre navegador e Node.js
- `JS-10.03` — Ausência de window, document e DOM no Node
- `JS-10.04` — Globais e APIs específicas de Node
- `JS-10.05` — process e variáveis de ambiente em nível introdutório
- `JS-10.06` — CommonJS scope-only globals
- `JS-10.07` — REPL, execução por CLI e organização por arquivos
- `JS-10.08` — Relação entre I/O não bloqueante e event loop no Node
