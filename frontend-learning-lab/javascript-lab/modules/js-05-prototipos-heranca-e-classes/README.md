# JS-05 — Protótipos, herança e classes

## Objetivo do módulo

Explicar o modelo de objetos do JavaScript, partindo de construtores e protótipos até classes modernas.

## Pré-requisitos

JS-04.

## Tópicos

- `JS-05.01` — Objetos, construtores e operador new
- `JS-05.02` — Cadeia de protótipos
- `JS-05.03` — Herança prototipal
- `JS-05.04` — Sintaxe de classes
- `JS-05.05` — Métodos de instância, getters e setters
- `JS-05.06` — Campos, métodos estáticos e elementos privados
- `JS-05.07` — extends, super e herança com classes
- `JS-05.08` — Relação entre classes e protótipos
