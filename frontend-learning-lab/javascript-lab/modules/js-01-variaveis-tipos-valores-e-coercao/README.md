# JS-01 — Variáveis, tipos, valores e coerção

## Objetivo do módulo

Construir a base semântica do JavaScript: declaração de valores, tipos primitivos, objetos, coerção, comparação e literais.

## Pré-requisitos

JS-00.

- `JS-01.04` — Objetos como valores de referência
- `JS-01.05` — typeof, truthy, falsy e coerção básica
- `JS-01.06` — Comparação com ===, == e Object.is
- `JS-01.07` — Literais e template literals
