# JS-00 — Ambiente, sintaxe e execução do JavaScript

## Objetivo do módulo

Estabelecer a base inicial da linguagem: o que é JavaScript, onde executa, como rodar scripts e quais regras sintáticas sustentam os módulos seguintes.

## Pré-requisitos

Nenhum.

## Tópicos

- `JS-00.01` — JavaScript, ECMAScript e runtimes
- `JS-00.02` — Navegador, console, REPL e execução por arquivo
- `JS-00.03` — Scripts, contexto inicial e noção de módulo
- `JS-00.04` — Sintaxe básica, comentários, instruções e blocos
- `JS-00.05` — Strict mode e efeitos gerais
