# DOM Lab

Este lab organiza o estudo do **Document Object Model (DOM)** como base para desenvolvimento frontend e fullstack. O foco é entender o documento como uma árvore viva no navegador e praticar seleção, navegação, leitura, alteração, criação, remoção, eventos, formulários, acessibilidade, segurança, performance e fundamentos de Shadow DOM.

## Objetivo

Consolidar o domínio do DOM como representação em memória do documento, manipulável por JavaScript, sem misturar o escopo deste lab com HTML, CSS, JavaScript puro, backend ou frameworks.

## Papel na formação

O DOM Lab conecta HTML, CSS e JavaScript no navegador. Ele prepara a base para interfaces dinâmicas, formulários, eventos, depuração de UI, projetos frontend e estudos posteriores de frameworks.

## Pré-requisitos

- HTML básico.
- CSS básico.
- JavaScript básico.
- Noção de navegador, DevTools e execução de scripts em páginas web.

## Como estudar

1. Siga os módulos em ordem.
2. Leia o `README.md` do módulo.
3. Use o `checklist.md` para acompanhar os tópicos e aulas.
4. Estude cada tópico pelo arquivo `lessons.md`.
5. Responda às perguntas em `assimilation-questions.md`.
6. Pratique dentro da pasta `exercises/`.
7. Ao finalizar o módulo, use `interview-questions.md` para revisar de forma integrada.
8. Desenvolva o projeto de conclusão do módulo em `docs/projects/`.

## Projetos

A pasta `docs/projects/` contém uma subpasta para cada módulo. Use essas pastas para construir projetos de conclusão de módulo, páginas demonstrativas e experimentos navegáveis.
