# DOM-00 — Fundamentos do documento, navegador e DOM

## Objetivo do módulo

Estabelecer o que é o DOM, como ele nasce a partir do HTML e como se relaciona com o documento renderizado, window e APIs do navegador.

## Pré-requisitos

HTML básico; CSS básico; sintaxe básica de JavaScript; noção geral de navegador.

## Progressão didática

Começa pela origem do documento e só depois conecta a árvore DOM ao ambiente do navegador e ao pipeline de renderização.

## Tópicos do módulo

- DOM-00.01 — HTML fonte e parsing do navegador
- DOM-00.02 — HTML fonte, DOM em memória e página renderizada
- DOM-00.03 — Estrutura da árvore DOM
- DOM-00.04 — DOM, window e APIs globais do navegador
- DOM-00.05 — DOM, CSSOM e pipeline de renderização
