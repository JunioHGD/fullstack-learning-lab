# Aulas — DOM-00.01: HTML fonte e parsing do navegador

## Objetivo do tópico

Entender o HTML como fonte serializada e o parsing como etapa de construção da árvore DOM.

---

## DOM-00.01.01 — HTML fonte como documento serializado

### Objetivo

Reconhecer o HTML escrito no arquivo como entrada textual usada pelo navegador para construir o documento.

### Pré-requisitos

HTML básico e noção geral de navegador.

### Conceitos a estudar

- HTML fonte como documento serializado
- Reconhecer o HTML escrito no arquivo como entrada textual usada pelo navegador para construir o documento.

### Resultado esperado

Explicar que o arquivo HTML é a fonte inicial, não a árvore viva manipulada em memória.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.

---

## DOM-00.01.02 — Parsing de HTML e construção inicial do DOM

### Objetivo

Compreender que o navegador interpreta o HTML e cria objetos DOM organizados em árvore.

### Pré-requisitos

DOM-00.01.01.

### Conceitos a estudar

- Parsing de HTML e construção inicial do DOM
- Compreender que o navegador interpreta o HTML e cria objetos DOM organizados em árvore.

### Resultado esperado

Reconhecer que o navegador pode construir, ajustar e normalizar a árvore DOM a partir do HTML fonte.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.
