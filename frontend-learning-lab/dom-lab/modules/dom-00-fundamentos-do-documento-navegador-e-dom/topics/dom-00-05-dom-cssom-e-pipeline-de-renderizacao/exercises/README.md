## Orientação

- Crie exercícios pequenos e focados.
- Priorize clareza do conceito antes de complexidade visual.
- Registre no código comentários úteis, não excessivos.
- Quando o exercício gerar uma página navegável, considere mover ou publicar uma versão em `docs/projects/`.
