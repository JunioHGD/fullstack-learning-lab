## DOM-00.04.01 — document, window e fronteiras do DOM

### Objetivo

Entender document como entrada para o DOM e window como objeto global do ambiente do navegador.

### Pré-requisitos

DOM-00.03.01.

### Conceitos a estudar

- document, window e fronteiras do DOM
- Entender document como entrada para o DOM e window como objeto global do ambiente do navegador.

### Resultado esperado

Explicar que DOM não é sinônimo de navegador inteiro nem de todas as Web APIs.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.
