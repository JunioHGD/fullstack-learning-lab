# Aulas — DOM-01.03: Document, Element e HTMLElement

## Objetivo do tópico

Diferenciar as interfaces centrais usadas para acessar documento e elementos HTML.

---

## DOM-01.03.01 — Document como ponto de entrada do DOM

### Objetivo

Entender document como objeto que representa a página carregada e dá acesso à árvore DOM.

### Pré-requisitos

DOM-01.02.01.

### Conceitos a estudar

- Document como ponto de entrada do DOM
- Entender document como objeto que representa a página carregada e dá acesso à árvore DOM.

### Resultado esperado

Usar corretamente o conceito de document como raiz operacional para seleção e criação futuras.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.
