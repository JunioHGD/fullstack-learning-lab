---

## DOM-01.04.01 — Especializações de elementos HTML

### Objetivo

Entender que inputs, forms, links, imagens e outros elementos possuem propriedades próprias.

### Pré-requisitos

DOM-01.03.02.

### Conceitos a estudar

- Especializações de elementos HTML
- Entender que inputs, forms, links, imagens e outros elementos possuem propriedades próprias.

### Resultado esperado

Reconhecer que nem todos os elementos HTML oferecem exatamente a mesma superfície de API.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.
