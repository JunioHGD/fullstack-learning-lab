# DOM-01 — Interfaces centrais e tipos de nós

## Objetivo do módulo

Formar a base mental do modelo de objetos do DOM antes do uso intensivo de métodos de seleção e manipulação.

## Pré-requisitos

DOM-00.

## Progressão didática

Apresenta primeiro as interfaces-base e depois objetos concretos, especializações HTML e coleções.

## Tópicos do módulo

- DOM-01.01 — Bases do modelo de interfaces
- DOM-01.02 — Node, tipos de nós e nodeType
- DOM-01.03 — Document, Element e HTMLElement
- DOM-01.04 — Interfaces HTML específicas
- DOM-01.05 — Coleções do DOM

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use cada pasta em `topics/` para estudar o conteúdo por tópico.
- Use `interview-questions.md` ao final para revisar o módulo de forma integrada.
- Use `docs/projects/dom-01-interfaces-centrais-e-tipos-de-nos/` para desenvolver o projeto de conclusão deste módulo.
