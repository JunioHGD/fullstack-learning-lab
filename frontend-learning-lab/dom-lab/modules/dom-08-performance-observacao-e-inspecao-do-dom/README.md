# DOM-08 — Performance, observação e inspeção do DOM

## Objetivo do módulo

Ensinar o DOM como estrutura com custo de renderização, superfície observável e objeto de depuração profissional.

## Pré-requisitos

DOM-03 a DOM-07; noções básicas de renderização e CSS.

## Progressão didática

Começa por custos e renderização, passa por atualização em lote e observação, e fecha com inspeção no DevTools.

## Tópicos do módulo

- DOM-08.01 — Custo do tamanho do DOM
- DOM-08.02 — Impacto de mudanças no pipeline visual
- DOM-08.03 — Reflow e layout thrashing
- DOM-08.04 — Atualização em lote
- DOM-08.05 — Observação de mutações
- DOM-08.06 — Inspeção da árvore DOM no DevTools
- DOM-08.07 — Depuração de eventos e mutações
- DOM-08.08 — Inspeção de acessibilidade e gargalos

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use cada pasta em `topics/` para estudar o conteúdo por tópico.
- Use `interview-questions.md` ao final para revisar o módulo de forma integrada.
- Use `docs/projects/dom-08-performance-observacao-e-inspecao-do-dom/` para desenvolver o projeto de conclusão deste módulo.
