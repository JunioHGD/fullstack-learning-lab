# DOM-07 — Semântica, acessibilidade e segurança de conteúdo

## Objetivo do módulo

Introduzir consequências de qualidade ao manipular o DOM: semântica, acessibilidade e segurança contra inserção insegura.

## Pré-requisitos

HTML Lab com semântica básica; DOM-03, DOM-05 e DOM-06.

## Progressão didática

Parte da acessibilidade derivada do DOM, passa por foco/semântica e fecha com segurança de conteúdo.

## Tópicos do módulo

- DOM-07.01 — DOM e árvore de acessibilidade
- DOM-07.02 — Semântica, ordem de fonte e foco
- DOM-07.03 — ARIA complementar
- DOM-07.04 — Atualizações acessíveis do DOM
- DOM-07.05 — Inserção segura de conteúdo
- DOM-07.06 — DOM XSS e Trusted Types

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use cada pasta em `topics/` para estudar o conteúdo por tópico.
- Use `interview-questions.md` ao final para revisar o módulo de forma integrada.
- Use `docs/projects/dom-07-semantica-acessibilidade-e-seguranca-de-conteudo/` para desenvolver o projeto de conclusão deste módulo.
