# DOM-03 — Conteúdo, atributos, classes, estilos e dataset

## Objetivo do módulo

Organizar as formas principais de ler e atualizar informações de elementos sem ainda alterar profundamente a estrutura da árvore.

## Pré-requisitos

DOM-01 e DOM-02.

## Progressão didática

Vai do texto seguro para HTML interpretado, atributos, propriedades, classes, estilos e metadados de elemento.

## Tópicos do módulo

- DOM-03.01 — Conteúdo textual
- DOM-03.02 — Conteúdo HTML
- DOM-03.03 — Atributos e propriedades
- DOM-03.04 — Classes e estilos inline
- DOM-03.05 — Atributos de dados e dataset
- DOM-03.06 — Critérios de escolha de APIs

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use cada pasta em `topics/` para estudar o conteúdo por tópico.
- Use `interview-questions.md` ao final para revisar o módulo de forma integrada.
- Use `docs/projects/dom-03-conteudo-atributos-classes-estilos-e-dataset/` para desenvolver o projeto de conclusão deste módulo.
