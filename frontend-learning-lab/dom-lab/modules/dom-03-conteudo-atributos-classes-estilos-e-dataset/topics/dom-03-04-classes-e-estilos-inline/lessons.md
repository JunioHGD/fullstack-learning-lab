# Aulas — DOM-03.04: Classes e estilos inline

## Objetivo do tópico

Controlar estado visual por classes e estilos inline.

---

## DOM-03.04.01 — Manipulação de classes com classList

### Objetivo

Usar classList para adicionar, remover, alternar e consultar classes.

### Pré-requisitos

DOM-03.03.02 e CSS básico.

### Conceitos a estudar

- Manipulação de classes com classList
- Usar classList para adicionar, remover, alternar e consultar classes.

### Resultado esperado

Controlar estados visuais por classes sem reescrever atributos manualmente.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.

## DOM-03.04.02 — Estilo inline via style

### Objetivo

Entender quando e como alterar estilos inline pelo DOM.

### Pré-requisitos

### Conceitos a estudar

- Estilo inline via style
- Entender quando e como alterar estilos inline pelo DOM.

### Resultado esperado

Diferenciar mudanças pontuais por style de estados visuais recorrentes por classe.

### Observações de estudo

Use exemplos pequenos, teste no navegador/DevTools quando aplicável e registre dúvidas antes de passar para a próxima aula.
