# DOM-09 — Shadow DOM e fundamentos de Web Components

## Objetivo do módulo

Fechar o DOM Lab com encapsulamento estrutural e composição de interface, sem substituir estudo amplo de Web Components.

## Pré-requisitos

DOM-01, DOM-04, DOM-05 e DOM-08.

## Progressão didática

Apresenta motivação, criação da shadow tree, encapsulamento, composição por slots, eventos e limites do módulo.

## Tópicos do módulo

- DOM-09.01 — Motivação do Shadow DOM
- DOM-09.02 — Criação e acesso à shadow tree
- DOM-09.03 — Encapsulamento de DOM e CSS
- DOM-09.04 — Slots e composição
- DOM-09.05 — Eventos em fronteiras de shadow tree
- DOM-09.06 — Templates com Shadow DOM
- DOM-09.07 — Declarative Shadow DOM e limites do módulo

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use cada pasta em `topics/` para estudar o conteúdo por tópico.
- Use `interview-questions.md` ao final para revisar o módulo de forma integrada.
- Use `docs/projects/dom-09-shadow-dom-e-fundamentos-de-web-components/` para desenvolver o projeto de conclusão deste módulo.
