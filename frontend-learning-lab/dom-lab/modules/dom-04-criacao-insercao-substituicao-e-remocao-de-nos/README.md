# DOM-04 — Criação, inserção, substituição e remoção de nós

## Objetivo do módulo

Cobrir a mutação estrutural do documento: criação, inserção, substituição, cópia, remoção e montagem de subárvores.

## Pré-requisitos

DOM-01, DOM-02 e DOM-03.

## Progressão didática

Começa com criação de nós, passa para inserção e transformação, e termina com fragmentos e templates.

## Tópicos do módulo

- DOM-04.01 — Criação de nós
- DOM-04.02 — Inserção básica
- DOM-04.03 — Inserções relativas e reordenação
- DOM-04.04 — Substituição de nós e filhos
- DOM-04.05 — Remoção de nós
- DOM-04.06 — Cópia de nós
- DOM-04.07 — Montagem em fragmentos
- DOM-04.08 — Templates HTML

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use cada pasta em `topics/` para estudar o conteúdo por tópico.
- Use `interview-questions.md` ao final para revisar o módulo de forma integrada.
- Use `docs/projects/dom-04-criacao-insercao-substituicao-e-remocao-de-nos/` para desenvolver o projeto de conclusão deste módulo.
