# DOM-06 — Formulários, submissão e validação nativa

## Objetivo do módulo

Cobrir o uso do DOM no subsistema de formulários: associação, coleta, submissão e validação no navegador.

## Pré-requisitos

HTML Lab com formulários básicos; DOM-03 e DOM-05.

## Progressão didática

Segue o domínio real dos formulários: estrutura, associação, acesso a dados, submissão, validação nativa e serialização.

## Tópicos do módulo

- DOM-06.01 — form e controles associados
- DOM-06.02 — Categorias e associação de controles
- DOM-06.03 — Acesso a controles e valores
- DOM-06.04 — Ciclo de submissão
- DOM-06.05 — Constraint validation nativa
- DOM-06.06 — APIs de validade
- DOM-06.07 — Validação customizada
- DOM-06.08 — Lista de entradas e serialização
