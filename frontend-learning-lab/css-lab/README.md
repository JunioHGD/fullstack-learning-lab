# CSS Lab

## Objetivo

Organizar o CSS Lab como uma trilha progressiva para dominar CSS moderno aplicado à criação de interfaces web profissionais, responsivas, acessíveis, organizadas e preparadas para projetos frontend/fullstack.

## Papel na formação

O CSS Lab faz parte do repositório `frontend-learning-lab` e funciona como a base técnica para transformar estruturas HTML em interfaces visuais. O lab cobre desde a aplicação inicial de CSS até layout avançado, responsividade, acessibilidade visual, organização de estilos, animações e recursos modernos.

## Pré-requisitos

- Conhecimento básico de HTML.
- Saber criar e editar arquivos `.html` e `.css`.
- Noção básica de elementos, atributos, classes e IDs.
- Familiaridade inicial com navegador e editor de código.

## Como estudar

1. Siga os módulos em ordem.
2. Leia o `README.md` do módulo para entender o escopo.
3. Use o `checklist.md` para acompanhar o progresso.
4. Estude cada tópico pelo arquivo `lessons.md`.
5. Responda `assimilation-questions.md`.
6. Pratique dentro da pasta `exercises/`.
7. Ao finalizar o módulo, desenvolva o projeto correspondente em `docs/projects/`.

## Relação com outros labs

- **HTML Lab:** fornece a estrutura sobre a qual o CSS será aplicado.
- **JavaScript Lab:** conecta-se posteriormente quando scripts alterarem classes, estados e estilos.
- **DOM Lab:** conecta-se pelo uso de seletores, classes e manipulação visual dinâmica.
- **Frontend Learning Lab:** o CSS Lab é um dos pilares junto com HTML, JavaScript e DOM.
- **Fullstack Projects Lab:** aplica o CSS em interfaces reais integradas a fluxos de aplicação.
- **Frameworks e componentização:** prepara para CSS Modules, componentes reutilizáveis, design tokens, bibliotecas de UI e frameworks frontend.
