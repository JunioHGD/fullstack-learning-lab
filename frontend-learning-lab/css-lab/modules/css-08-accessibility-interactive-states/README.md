# CSS-08 — Acessibilidade e Estados Interativos

## Objetivo

Garantir que estilos sejam legíveis, navegáveis, inclusivos e claros em estados interativos.

## Pré-requisitos

Cores, tipografia, pseudo-classes, organização CSS e noções de componentes.

## Tópicos

| Código | Tópico | Aulas | Pasta |
|---|---|---:|---|
| CSS-08.01 | Contraste e legibilidade | 2 | `css-08-01-contrast-legibility` |
| CSS-08.02 | Foco visível | 2 | `css-08-02-visible-focus` |
| CSS-08.03 | Estados de interação | 2 | `css-08-03-interaction-states` |
| CSS-08.04 | Melhorias progressivas | 1 | `css-08-04-progressive-enhancement` |

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisão integrada e preparação para entrevistas.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/css-08-accessibility-interactive-states/`.
