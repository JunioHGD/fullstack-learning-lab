# Interview Questions — CSS-10 — CSS Moderno: Container Queries, Cascade Layers e Nesting

Perguntas integradas para revisar o módulo e treinar explicações técnicas em entrevistas.

## CSS-10.01 — Container Queries

1. Explique o objetivo deste tópico: Aplicar estilos com base no tamanho do container e não apenas da viewport.
2. Como você explicaria `Limites das media queries e lógica de container` para alguém que está começando?
3. Em que situação prática você aplicaria `Container type, name e @container`?

## CSS-10.02 — Cascade Layers

1. Explique o objetivo deste tópico: Controlar a precedência de estilos usando camadas de cascata.
2. Como você explicaria `Fundamentos de @layer` para alguém que está começando?
3. Em que situação prática você aplicaria `Camadas base, components e utilities`?

## CSS-10.03 — Nesting

1. Explique o objetivo deste tópico: Usar aninhamento nativo de seletores com legibilidade e controle de especificidade.
2. Como você explicaria `Sintaxe de nesting e uso do &` para alguém que está começando?
3. Em que situação prática você aplicaria `Cuidados com profundidade e especificidade`?
