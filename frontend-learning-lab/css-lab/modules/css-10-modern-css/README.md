# CSS-10 — CSS Moderno: Container Queries, Cascade Layers e Nesting

## Objetivo

Explorar recursos recentes que aumentam encapsulamento, previsibilidade e organização em componentes modernos.

## Pré-requisitos

Cascata, especificidade, responsividade, organização CSS, Flexbox e Grid.
