# CSS-01.03 — Especificidade

## Módulo

CSS-01 — Seletores, Especificidade e Cascata

## Objetivo do tópico

Entender o peso dos seletores quando regras diferentes competem pelo mesmo elemento.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 01.03.01 | Cálculo de especificidade | Aprender como o navegador compara o peso dos seletores. |
| 01.03.02 | Conflitos comuns de especificidade | Reconhecer conflitos gerados por seletores muito fortes ou mal combinados. |

- `lessons.md`: explicação dos conceitos e das aulas do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
