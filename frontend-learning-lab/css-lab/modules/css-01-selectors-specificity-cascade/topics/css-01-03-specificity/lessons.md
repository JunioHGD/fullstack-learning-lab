# Lessons — CSS-01.03 — Especificidade

## Objetivo do tópico

Entender o peso dos seletores quando regras diferentes competem pelo mesmo elemento.

## 01.03.01 — Cálculo de especificidade

### Objetivo

Aprender como o navegador compara o peso dos seletores.

### Pré-requisitos

Seletores básicos, combinadores e pseudo-seletores introdutórios.

### Conceitos que devem ficar claros

- O que significa **Cálculo de especificidade** dentro deste tópico.
- Como este conceito se conecta ao objetivo do tópico: Entender o peso dos seletores quando regras diferentes competem pelo mesmo elemento.
- Quais problemas práticos esse conceito ajuda a resolver.
- Quais erros comuns devem ser evitados ao aplicar esse conceito.

### Resultado esperado

Explicar por que uma regra vence outra em conflitos simples.
