# CSS-01.01 — Seletores básicos e combinadores

## Módulo

CSS-01 — Seletores, Especificidade e Cascata

## Objetivo do tópico

Aprender a selecionar elementos por tipo, classe, ID, atributo e relação estrutural.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 01.01.01 | Seletores de tipo, classe, ID e universal | Dominar os seletores mais usados no início do CSS. |
| 01.01.02 | Seletores de atributo | Selecionar elementos com base na presença ou valor de atributos. |
| 01.01.03 | Combinadores estruturais | Entender combinadores descendente, filho direto, irmão adjacente e irmãos gerais. |

## Arquivos do tópico

- `lessons.md`: explicação dos conceitos e das aulas do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios práticos, incluindo arquivos de código.
