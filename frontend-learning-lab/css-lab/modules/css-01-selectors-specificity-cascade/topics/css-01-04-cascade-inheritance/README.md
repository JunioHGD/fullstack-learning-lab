# CSS-01.04 — Cascata e herança

## Módulo

CSS-01 — Seletores, Especificidade e Cascata

## Objetivo do tópico

Entender como ordem, origem, importância, especificidade e herança formam o estilo final.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 01.04.01 | Cascata e ordem das regras | Compreender a influência da ordem e da origem das regras no resultado final. |
| 01.04.02 | Herança e valores herdáveis | Entender quais propriedades podem ser herdadas e como isso afeta elementos filhos. |
