# CSS-01 — Seletores, Especificidade e Cascata

## Objetivo

Ensinar a selecionar elementos corretamente e entender como o navegador decide quais estilos prevalecem em conflitos.

## Pré-requisitos

CSS-00 completo, HTML básico e domínio inicial dos atributos class e id.
