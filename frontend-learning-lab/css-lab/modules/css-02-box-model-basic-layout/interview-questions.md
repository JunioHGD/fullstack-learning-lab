# Interview Questions — CSS-02 — Box Model e Layout Básico

Perguntas integradas para revisar o módulo e treinar explicações técnicas em entrevistas.

## CSS-02.01 — Box Model

1. Explique o objetivo deste tópico: Compreender que todo elemento visual é uma caixa com conteúdo, padding, border e margin.
2. Como você explicaria `Content, padding, border e margin` para alguém que está começando?
3. Em que situação prática você aplicaria `Box sizing e cálculo de tamanho`?

## CSS-02.02 — Display e fluxo normal

1. Explique o objetivo deste tópico: Entender o comportamento padrão dos elementos no fluxo do documento.
2. Como você explicaria `Fluxo normal do documento` para alguém que está começando?
3. Em que situação prática você aplicaria `Display block, inline, inline-block e none`?

## CSS-02.03 — Positioning

1. Explique o objetivo deste tópico: Controlar deslocamento e posicionamento de elementos fora do comportamento padrão.
2. Como você explicaria `Position static, relative e absolute` para alguém que está começando?
3. Em que situação prática você aplicaria `Position fixed, sticky e z-index inicial`?

## CSS-02.04 — Width, height e overflow

1. Explique o objetivo deste tópico: Controlar dimensões mínimas, máximas e comportamento de conteúdo excedente.
2. Como você explicaria `Width, height, min e max` para alguém que está começando?
3. Em que situação prática você aplicaria `Overflow e conteúdo excedente`?
