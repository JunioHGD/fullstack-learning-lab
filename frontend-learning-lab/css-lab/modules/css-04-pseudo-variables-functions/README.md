# CSS-04 — Pseudo-classes, Pseudo-elementos e Variáveis CSS

## Objetivo

Expandir seletores especiais e introduzir custom properties e funções CSS para reutilização e valores dinâmicos.

## Pré-requisitos

Seletores básicos, estilização visual, cascata e box model.
