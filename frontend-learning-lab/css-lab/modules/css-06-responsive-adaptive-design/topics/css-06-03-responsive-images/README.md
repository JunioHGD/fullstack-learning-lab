# CSS-06.03 — Imagens responsivas

## Módulo

CSS-06 — Responsividade e Design Adaptativo

## Objetivo do tópico

Evitar que imagens quebrem layouts e compreender estratégias básicas de adaptação.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 06.03.01 | Imagens flexíveis com CSS | Aplicar max-width, height auto, object-fit e background-size. |
| 06.03.02 | Srcset, sizes e picture como conexão com HTML | Reconhecer srcset, sizes e picture como suporte HTML à responsividade de imagens. |
