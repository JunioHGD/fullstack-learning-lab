## 06.03.01 — Imagens flexíveis com CSS

### Objetivo

Aplicar max-width, height auto, object-fit e background-size.

### Pré-requisitos

Layouts fluidos e box model.

### Conceitos que devem ficar claros

- O que significa **Imagens flexíveis com CSS** dentro deste tópico.
- Como este conceito se conecta ao objetivo do tópico: Evitar que imagens quebrem layouts e compreender estratégias básicas de adaptação.
- Quais problemas práticos esse conceito ajuda a resolver.
- Quais erros comuns devem ser evitados ao aplicar esse conceito.

### Resultado esperado

Adaptar imagens ao espaço sem distorção ou overflow indevido.
