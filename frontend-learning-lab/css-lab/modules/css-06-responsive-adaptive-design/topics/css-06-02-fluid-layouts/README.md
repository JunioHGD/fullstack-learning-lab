# CSS-06.02 — Layouts fluidos

## Módulo

CSS-06 — Responsividade e Design Adaptativo

## Objetivo do tópico

Criar layouts que se ajustam naturalmente ao espaço disponível.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 06.02.01 | Percentuais e unidades relativas em layouts | Usar porcentagem, em, rem, vw e vh em estruturas fluidas. |
| 06.02.02 | Largura máxima e valores fluidos com clamp | Combinar max-width, width e clamp() para layouts mais adaptáveis. |

- `lessons.md`: explicação dos conceitos e das aulas do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios práticos, incluindo arquivos de código.
