## 06.04.01 — Responsivo, adaptativo e decisões de layout

### Objetivo

Diferenciar design responsivo e adaptativo em decisões práticas de layout.

### Pré-requisitos

Media queries, layouts fluidos e imagens responsivas.

### Conceitos que devem ficar claros

- O que significa **Responsivo, adaptativo e decisões de layout** dentro deste tópico.
- Como este conceito se conecta ao objetivo do tópico: Comparar estratégias fluidas e estratégias baseadas em pontos de ruptura.
- Quais problemas práticos esse conceito ajuda a resolver.
- Quais erros comuns devem ser evitados ao aplicar esse conceito.

### Resultado esperado

Justificar uma estratégia de adaptação conforme o tipo de interface.
