# CSS-06 — Responsividade e Design Adaptativo

## Objetivo

Adaptar interfaces a diferentes telas com media queries, layout fluido, imagens responsivas e abordagem mobile-first.

## Pré-requisitos

Flexbox, Grid, unidades relativas, box model e estilização visual.

| CSS-06.03 | Imagens responsivas | 2 | `css-06-03-responsive-images` |
