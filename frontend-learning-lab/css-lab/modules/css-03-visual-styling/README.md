# CSS-03 — Estilização Visual: Cores, Tipografia e Fundos

## Objetivo

Desenvolver a base visual para cores, unidades, tipografia, fundos, gradientes, bordas e sombras.

## Pré-requisitos

Box model, display, seletores, cascata e herança.

## Tópicos

| Código | Tópico | Aulas | Pasta |
|---|---|---:|---|
| CSS-03.01 | Cores e unidades | 2 | `css-03-01-colors-units` |
| CSS-03.02 | Tipografia | 2 | `css-03-02-typography` |
| CSS-03.03 | Backgrounds e gradientes | 2 | `css-03-03-backgrounds-gradients` |
| CSS-03.04 | Borders e sombreados | 1 | `css-03-04-borders-shadows` |

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisão integrada e preparação para entrevistas.
- Acesse cada tópico em `topics/`.
