# CSS-03.01 — Cores e unidades

## Módulo

CSS-03 — Estilização Visual: Cores, Tipografia e Fundos

## Objetivo do tópico

Entender formatos de cor e unidades CSS fundamentais.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 03.01.01 | Formatos de cor no CSS | Conhecer cores nomeadas, hexadecimal, rgb, rgba, hsl e hsla. |
| 03.01.02 | Unidades absolutas e relativas | Diferenciar px, porcentagem, em, rem, vw e vh. |
