# Interview Questions — CSS-05 — Layout Avançado: Flexbox e CSS Grid

Perguntas integradas para revisar o módulo e treinar explicações técnicas em entrevistas.

## CSS-05.01 — Flexbox

1. Explique o objetivo deste tópico: Construir layouts em um eixo com controle de alinhamento, direção e flexibilidade dos itens.
2. Como você explicaria `Modelo flex e eixos` para alguém que está começando?
3. Em que situação prática você aplicaria `Crescimento, encolhimento e quebra de itens`?

## CSS-05.02 — Grid Layout

1. Explique o objetivo deste tópico: Construir layouts bidimensionais com linhas, colunas e posicionamento de itens.
2. Como você explicaria `Modelo grid, linhas, colunas e gap` para alguém que está começando?
3. Em que situação prática você aplicaria `Posicionamento de itens no Grid`?

## CSS-05.03 — Layouts compostos

1. Explique o objetivo deste tópico: Combinar Flexbox e Grid em estruturas comuns de interface.
2. Como você explicaria `Quando usar Flexbox ou Grid` para alguém que está começando?
3. Em que situação prática você aplicaria `Composição de headers, cards e painéis`?
