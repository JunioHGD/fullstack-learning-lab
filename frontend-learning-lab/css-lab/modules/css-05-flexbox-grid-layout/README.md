# CSS-05 — Layout Avançado: Flexbox e CSS Grid

## Objetivo

Ensinar layouts modernos unidimensionais e bidimensionais com Flexbox, Grid e composições combinadas.

## Pré-requisitos

Box model, display, fluxo normal, position, dimensões e overflow.

| CSS-05.02 | Grid Layout | 3 | `css-05-02-grid-layout` |
