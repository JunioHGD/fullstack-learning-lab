# CSS-05.02 — Grid Layout

## Módulo

CSS-05 — Layout Avançado: Flexbox e CSS Grid

## Objetivo do tópico

Construir layouts bidimensionais com linhas, colunas e posicionamento de itens.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 05.02.01 | Modelo grid, linhas, colunas e gap | Entender container grid, linhas, colunas, gap e unidade fr. |
| 05.02.02 | Grid template e repetição de colunas | Usar grid-template-columns, grid-template-rows e repeat(). |
| 05.02.03 | Posicionamento de itens no Grid | Usar grid-column, grid-row e span para posicionar itens. |
