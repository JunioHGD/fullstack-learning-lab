# CSS-05.01 — Flexbox

## Módulo

CSS-05 — Layout Avançado: Flexbox e CSS Grid

## Objetivo do tópico

Construir layouts em um eixo com controle de alinhamento, direção e flexibilidade dos itens.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 05.01.01 | Modelo flex e eixos | Entender container flex, itens flex, eixo principal e eixo cruzado. |
| 05.01.02 | Direção, alinhamento e distribuição | Usar flex-direction, justify-content, align-items, align-content e gap. |
| 05.01.03 | Crescimento, encolhimento e quebra de itens | Controlar flex-grow, flex-shrink, flex-basis, flex e flex-wrap. |

## Arquivos do tópico

- `lessons.md`: explicação dos conceitos e das aulas do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios práticos, incluindo arquivos de código.
