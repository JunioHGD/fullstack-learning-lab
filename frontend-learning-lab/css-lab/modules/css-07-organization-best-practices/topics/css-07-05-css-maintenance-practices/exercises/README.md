# Exercises — CSS-07.05 — Melhores práticas gerais

Esta pasta deve reunir exercícios práticos relacionados ao tópico.

## Módulo

CSS-07 — Organização e Boas Práticas de CSS

## Aulas relacionadas

- 07.05.01 — Especificidade saudável e uso responsável de important

## Sugestão de organização

Crie uma subpasta para cada exercício prático:

```txt
exercises/
├── 01-nome-do-exercicio/
│   ├── index.html
│   └── style.css
└── 02-nome-do-exercicio/
    ├── index.html
    └── style.css
```

## Critérios de prática

- Resolver exercícios pequenos antes de projetos maiores.
- Registrar dúvidas no próprio exercício ou no checklist do módulo.
- Priorizar clareza, organização e aplicação correta do conceito.
