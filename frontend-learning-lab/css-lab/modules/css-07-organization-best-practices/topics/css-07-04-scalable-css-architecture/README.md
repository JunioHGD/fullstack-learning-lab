# CSS-07.04 — Arquitetura escalável

## Módulo

CSS-07 — Organização e Boas Práticas de CSS

## Objetivo do tópico

Introduzir princípios para manter CSS previsível em projetos de médio porte.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 07.04.01 | CSS modular e separação de responsabilidades | Entender separação entre estilos globais, layout, componentes e utilitários. |
| 07.04.02 | Tokens visuais simples | Organizar cores, espaçamentos e tipografia como valores reutilizáveis. |

## Arquivos do tópico

- `lessons.md`: explicação dos conceitos e das aulas do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios práticos, incluindo arquivos de código.
