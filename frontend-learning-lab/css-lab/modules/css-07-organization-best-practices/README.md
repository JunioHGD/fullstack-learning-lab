# CSS-07 — Organização e Boas Práticas de CSS

## Objetivo

Organizar CSS para projetos maiores com reset/normalize, estrutura de arquivos, BEM, arquitetura escalável e manutenção.

## Pré-requisitos

Fundamentos de CSS, seletores, cascata, box model, layout, responsividade e estilização visual.

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisão integrada e preparação para entrevistas.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/css-07-organization-best-practices/`.
