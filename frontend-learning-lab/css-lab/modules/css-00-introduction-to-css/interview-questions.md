# Interview Questions — CSS-00 — Introdução ao CSS

Perguntas integradas para revisar o módulo e treinar explicações técnicas em entrevistas.

## CSS-00.01 — O que é CSS e como ele funciona

1. Explique o objetivo deste tópico: Entender o papel do CSS como linguagem de apresentação visual aplicada sobre documentos HTML.
2. Como você explicaria `Papel do CSS na Web` para alguém que está começando?
3. Em que situação prática você aplicaria `CSS como camada de apresentação`?

## CSS-00.02 — Formas de aplicar CSS

1. Explique o objetivo deste tópico: Conhecer as formas principais de inserir CSS em um documento.
2. Como você explicaria `CSS inline, interno e externo` para alguém que está começando?
3. Em que situação prática você aplicaria `CSS inline, interno e externo`?

## CSS-00.03 — Sintaxe básica de CSS

1. Explique o objetivo deste tópico: Aprender a estrutura mínima de uma regra CSS e a relação entre seletor, propriedade e valor.
2. Como você explicaria `Regras, seletores, propriedades e valores` para alguém que está começando?
3. Em que situação prática você aplicaria `Escopo inicial das regras e comentários`?

## CSS-00.04 — Introdução ao DevTools para CSS

1. Explique o objetivo deste tópico: Usar o navegador para inspecionar elementos e observar estilos aplicados.
2. Como você explicaria `Inspeção básica de estilos no navegador` para alguém que está começando?
3. Em que situação prática você aplicaria `Inspeção básica de estilos no navegador`?
