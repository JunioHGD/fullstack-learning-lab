# Lessons — CSS-00.02 — Formas de aplicar CSS

## Objetivo do tópico

Conhecer as formas principais de inserir CSS em um documento.

## 00.02.01 — CSS inline, interno e externo

### Objetivo

Diferenciar estilos inline, bloco style e arquivo CSS externo.

### Pré-requisitos

Compreensão básica do papel do CSS.

### Conceitos que devem ficar claros

- O que significa **CSS inline, interno e externo** dentro deste tópico.
- Como este conceito se conecta ao objetivo do tópico: Conhecer as formas principais de inserir CSS em um documento.
- Quais problemas práticos esse conceito ajuda a resolver.
- Quais erros comuns devem ser evitados ao aplicar esse conceito.

### Resultado esperado

Identificar as três formas de aplicar CSS e reconhecer a mais adequada para manutenção.
