# CSS-00.01 — O que é CSS e como ele funciona

## Módulo

CSS-00 — Introdução ao CSS

## Objetivo do tópico

Entender o papel do CSS como linguagem de apresentação visual aplicada sobre documentos HTML.

## Aulas

| Código | Aula | Objetivo |
|---|---|---|
| 00.01.01 | Papel do CSS na Web | Compreender o que é CSS e por que ele existe na construção de interfaces. |
| 00.01.02 | CSS como camada de apresentação | Entender como o CSS atua sobre elementos HTML sem substituir a estrutura semântica. |
