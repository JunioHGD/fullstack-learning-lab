### Objetivo

### Pré-requisitos

HTML básico e noção de página web.

### Conceitos que devem ficar claros

- O que significa **Papel do CSS na Web** dentro deste tópico.
- Como este conceito se conecta ao objetivo do tópico: Entender o papel do CSS como linguagem de apresentação visual aplicada sobre documentos HTML.
- Quais problemas práticos esse conceito ajuda a resolver.
- Quais erros comuns devem ser evitados ao aplicar esse conceito.

### Resultado esperado

Explicar o papel do CSS na separação entre conteúdo, estrutura e apresentação.
