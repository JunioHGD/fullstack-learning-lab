## 00.04.01 — Inspeção básica de estilos no navegador

### Objetivo

Conhecer o painel de inspeção e identificar regras CSS aplicadas a um elemento.

### Pré-requisitos

Sintaxe básica de CSS.

### Conceitos que devem ficar claros

- O que significa **Inspeção básica de estilos no navegador** dentro deste tópico.
- Como este conceito se conecta ao objetivo do tópico: Usar o navegador para inspecionar elementos e observar estilos aplicados.
- Quais problemas práticos esse conceito ajuda a resolver.
- Quais erros comuns devem ser evitados ao aplicar esse conceito.

### Resultado esperado

Abrir o DevTools, selecionar um elemento e reconhecer estilos ativos e sobrescritos.
