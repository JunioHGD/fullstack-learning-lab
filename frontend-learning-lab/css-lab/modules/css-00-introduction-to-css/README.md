# CSS-00 — Introdução ao CSS

## Objetivo

Apresentar o papel do CSS na Web, sua relação com HTML, suas formas de aplicação, sua sintaxe básica e o uso inicial do DevTools.

## Pré-requisitos

Noções básicas de HTML, edição de arquivos web e uso básico de navegador/editor de código.

## Tópicos

| Código | Tópico | Aulas | Pasta |
|---|---|---:|---|
| CSS-00.01 | O que é CSS e como ele funciona | 2 | `css-00-01-css-role` |
| CSS-00.02 | Formas de aplicar CSS | 1 | `css-00-02-applying-css` |
| CSS-00.03 | Sintaxe básica de CSS | 2 | `css-00-03-css-syntax` |
| CSS-00.04 | Introdução ao DevTools para CSS | 1 | `css-00-04-css-devtools-intro` |

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisão integrada e preparação para entrevistas.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/css-00-introduction-to-css/`.
