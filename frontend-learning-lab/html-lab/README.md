# HTML Lab

## Objetivo

Estruturar uma trilha completa de aulas para consolidar HTML como a base estrutural da Web, com foco em documentos bem formados, semântica, acessibilidade, conteúdo textual, links, mídia, formulários, metadados, validação e preparação para CSS, JavaScript, DOM e projetos fullstack.

## Papel na formação

O HTML Lab é a primeira fundação técnica do frontend-learning-lab. Ele prepara o estudante para construir a estrutura das páginas que depois serão estilizadas no CSS Lab, manipuladas no DOM/JavaScript Lab e integradas a fluxos fullstack por meio de formulários e páginas reais.

## Pré-requisitos

- Noções básicas de uso de computador e arquivos.
- Editor de código instalado.
- Navegador moderno para testar páginas HTML.
- Git e GitHub em nível básico são recomendados para versionar a evolução do Lab.

## Módulos

- **HTML-00 — Introdução e Estrutura Básica**: Apresentar o papel do HTML na Web, a estrutura mínima de um documento e os conceitos básicos de elementos, tags, atributos e aninhamento.
- **HTML-01 — Conteúdo de Texto e Hiperlinks**: Ensinar a marcar conteúdo textual, listas, links, imagens simples e caminhos de arquivos com semântica e clareza.
- **HTML-02 — Estrutura Semântica de Página**: Ensinar a organizar páginas em regiões semânticas com elementos estruturais e landmarks.
- **HTML-03 — Imagens e Mídia**: Aprofundar o uso de imagens, figuras, áudio, vídeo, iframes e templates em HTML.
- **HTML-04 — Formulários e Controles de Usuário**: Criar formulários HTML completos, acessíveis, validáveis e preparados para integração futura com backend.
- **HTML-05 — Metadados, Recursos Externos e SEO**: Configurar documentos HTML para publicação, recursos externos, SEO básico, integração CSS/JS e validação.
- **HTML-06 — Acessibilidade e Boas Práticas**: Consolidar HTML profissional com acessibilidade, navegação por teclado, foco, semântica, validação e segurança básica.
- **HTML-07 — Elementos Interativos e Avançados**: Apresentar elementos HTML menos comuns, dados tabulares, tempo, interatividade nativa e pontes para estudos posteriores.

## Como estudar

1. Siga os módulos em ordem.
2. Leia o `README.md` do módulo antes de iniciar.
3. Use o `checklist.md` como mapa de progresso.
4. Estude cada tópico pelo arquivo `lessons.md`.
5. Responda o `assimilation-questions.md`.
6. Resolva ou crie exercícios na pasta `exercises/`.
7. Ao concluir o módulo, desenvolva um projeto em `docs/projects/`.

## Projetos de conclusão

A pasta `docs/projects/` reúne uma subpasta por módulo. Use cada uma delas para criar projetos de conclusão de módulo, páginas demonstrativas ou atividades publicáveis no navegador.
