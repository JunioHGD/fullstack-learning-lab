# 00.03 — Tags fundamentais do documento

## Módulo

HTML-00 — Introdução e Estrutura Básica

## Objetivo do tópico

Aprofundar as tags essenciais do documento e os metadados iniciais.

## Aulas deste tópico

- **00.03.01 — Doctype, html, head e body**: Compreender a função dos elementos estruturais globais do documento.
- **00.03.02 — Title, meta charset, viewport e lang**: Configurar título, codificação, viewport e idioma do documento.
