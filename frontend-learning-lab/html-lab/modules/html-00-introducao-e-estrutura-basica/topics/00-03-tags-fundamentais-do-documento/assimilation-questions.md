# Perguntas de assimilação — 00.03 — Tags fundamentais do documento
Use estas perguntas para verificar se você realmente entendeu os conceitos do tópico.

## 00.03.01 — Doctype, html, head e body
1. Qual é a ideia central da aula **Doctype, html, head e body**?
2. Que problema essa aula ajuda a resolver ao escrever HTML?
3. Como você explicaria este objetivo com suas próprias palavras: Compreender a função dos elementos estruturais globais do documento.
4. Qual seria um exemplo prático de aplicação desse conceito em uma página real?
5. O que você precisa demonstrar para alcançar este resultado esperado: Explica a função de doctype, html, head e body dentro do documento.
6. Quais pré-requisitos precisam estar claros antes desta aula: Saber reconhecer a estrutura mínima de um HTML.

## 00.03.02 — Title, meta charset, viewport e lang
1. Qual é a ideia central da aula **Title, meta charset, viewport e lang**?
2. Que problema essa aula ajuda a resolver ao escrever HTML?
3. Como você explicaria este objetivo com suas próprias palavras: Configurar título, codificação, viewport e idioma do documento.
4. Qual seria um exemplo prático de aplicação desse conceito em uma página real?
5. O que você precisa demonstrar para alcançar este resultado esperado: Consegue configurar metadados iniciais e idioma de uma página HTML.
6. Quais pré-requisitos precisam estar claros antes desta aula: Conhecer head e html como partes estruturais do documento.
