# Aulas — 00.01 — O que é HTML e seu papel na Web
## Objetivo do tópico

Compreender HTML como linguagem de marcação responsável pela estrutura e significado do conteúdo web.

## Aulas

### 00.01.01 — O papel do HTML na Web

#### Conceito central

Esta aula trabalha o conceito de **O papel do HTML na Web** dentro do tópico **O que é HTML e seu papel na Web**. O foco é explicar o que é HTML e como ele estrutura o conteúdo na Web..

#### Objetivo da aula

Explicar o que é HTML e como ele estrutura o conteúdo na Web.

#### Pré-requisitos

Nenhum.

#### Resultado esperado

Consegue diferenciar HTML, CSS, JavaScript e DOM em nível introdutório.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Inicia o Lab pelo papel conceitual do HTML antes de qualquer sintaxe ou código.
