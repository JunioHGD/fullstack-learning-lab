# 00.05 — Atributos globais

## Módulo

HTML-00 — Introdução e Estrutura Básica

## Objetivo do tópico

Entender atributos reutilizáveis em diferentes elementos HTML.

## Aulas deste tópico

- **00.05.01 — Id, class, title e style**: Reconhecer atributos globais comuns e seus limites de uso.
- **00.05.02 — Lang, role e atributos de acessibilidade básicos**: Introduzir atributos relacionados a idioma e acessibilidade sem aprofundar ARIA.

- `assimilation-questions.md`: perguntas para fixação.
- `exercises/`: exercícios práticos, incluindo código quando necessário.
