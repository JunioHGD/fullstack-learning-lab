# Aulas — 00.04 — Elementos de texto simples
## Objetivo do tópico

Conhecer elementos básicos para marcar textos simples e separações pontuais.

## Aulas

### 00.04.01 — Parágrafo, span, quebra e separação temática

#### Conceito central

Esta aula trabalha o conceito de **Parágrafo, span, quebra e separação temática** dentro do tópico **Elementos de texto simples**. O foco é usar p, span, br e hr em contextos adequados..

#### Objetivo da aula

Usar p, span, br e hr em contextos adequados.

#### Pré-requisitos

Conhecer a estrutura básica do documento.

#### Resultado esperado

Reconhece quando usar parágrafo, conteúdo inline, quebra pontual e separação temática.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Introduz conteúdo visível simples antes de avançar para títulos, listas e links no módulo seguinte.
