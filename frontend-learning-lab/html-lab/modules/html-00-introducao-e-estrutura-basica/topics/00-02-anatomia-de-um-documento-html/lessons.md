# Aulas — 00.02 — Anatomia de um documento HTML
## Objetivo do tópico

Conhecer a estrutura mínima de um arquivo HTML válido.

## Aulas

### 00.02.01 — Estrutura mínima de um documento HTML

#### Conceito central

Esta aula trabalha o conceito de **Estrutura mínima de um documento HTML** dentro do tópico **Anatomia de um documento HTML**. O foco é identificar e montar a estrutura mínima com doctype, html, head e body..

#### Objetivo da aula

Identificar e montar a estrutura mínima com doctype, html, head e body.

#### Pré-requisitos

Compreender o papel geral do HTML.

#### Resultado esperado

Consegue criar um arquivo HTML completo e válido em sua forma mínima.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Depois de entender o papel do HTML, o aluno passa a reconhecer a forma mínima de um documento.
