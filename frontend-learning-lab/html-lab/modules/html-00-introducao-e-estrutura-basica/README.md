# HTML-00 — Introdução e Estrutura Básica

## Objetivo do módulo

Apresentar o papel do HTML na Web, a estrutura mínima de um documento e os conceitos básicos de elementos, tags, atributos e aninhamento.

## Pré-requisitos

Nenhum. Este módulo é o ponto de entrada do HTML Lab.

## Tópicos do módulo

- **00.01 — O que é HTML e seu papel na Web**: Compreender HTML como linguagem de marcação responsável pela estrutura e significado do conteúdo web.
- **00.02 — Anatomia de um documento HTML**: Conhecer a estrutura mínima de um arquivo HTML válido.
- **00.03 — Tags fundamentais do documento**: Aprofundar as tags essenciais do documento e os metadados iniciais.
- **00.04 — Elementos de texto simples**: Conhecer elementos básicos para marcar textos simples e separações pontuais.
- **00.05 — Atributos globais**: Entender atributos reutilizáveis em diferentes elementos HTML.
- **00.06 — Aninhamento e estrutura em árvore**: Compreender hierarquia, aninhamento e relação entre elementos.
