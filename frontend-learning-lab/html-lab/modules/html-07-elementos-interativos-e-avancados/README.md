# HTML-07 — Elementos Interativos e Avançados

## Objetivo do módulo

Apresentar elementos HTML menos comuns, dados tabulares, tempo, interatividade nativa e pontes para estudos posteriores.

## Pré-requisitos

HTML-00 a HTML-06.

## Tópicos do módulo

- **07.01 — Tabelas de dados**: Marcar dados tabulares com estrutura semântica e acessível.
- **07.02 — Elementos de tempo**: Marcar datas e horários de forma semântica.
- **07.03 — Elementos interativos adicionais**: Conhecer elementos nativos para revelação, diálogos, progresso e medição.
- **07.04 — Elementos de mídia dinâmica**: Revisitar iframe e script como elementos de integração.
- **07.05 — Templates avançados**: Revisar template como base para estudos futuros de DOM e frameworks.
- **07.06 — Boas práticas avançadas**: Refinar organização, clareza e manutenção de HTML em páginas maiores.
- **07.07 — Próximos passos no HTML**: Conectar a conclusão do HTML Lab aos próximos estudos.

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
