# Aulas — 07.05 — Templates avançados
## Objetivo do tópico

Revisar template como base para estudos futuros de DOM e frameworks.

## Aulas

### 07.05.01 — Template e preparação para componentes

#### Conceito central

Esta aula trabalha o conceito de **Template e preparação para componentes** dentro do tópico **Templates avançados**. O foco é entender template como marcação reutilizável para clonagem futura..

#### Objetivo da aula

Entender template como marcação reutilizável para clonagem futura.

#### Pré-requisitos

Conhecer template introdutório e estrutura semântica.

#### Resultado esperado

Consegue explicar por que template se conecta a DOM, JavaScript e frameworks.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Templates aparecem novamente no fim do Lab como ponte natural para DOM/JS.
