# Aulas — 07.07 — Próximos passos no HTML
## Objetivo do tópico

Conectar a conclusão do HTML Lab aos próximos estudos.

## Aulas

### 07.07.01 — Revisão final e ponte para outros Labs

#### Conceito central

Esta aula trabalha o conceito de **Revisão final e ponte para outros Labs** dentro do tópico **Próximos passos no HTML**. O foco é revisar a base de HTML profissional e preparar a continuidade da trilha..

#### Objetivo da aula

Revisar a base de HTML profissional e preparar a continuidade da trilha.

#### Pré-requisitos

Ter concluído todos os módulos anteriores.

#### Resultado esperado

Consegue avaliar se uma página HTML está pronta para receber CSS, JS, DOM e integração backend.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Fecha o Lab com uma visão de transição para CSS, DOM, JavaScript, backend e frameworks.
