Revisitar iframe e script como elementos de integração.

## Aulas

### 07.04.01 — Iframe e script revisitados

#### Conceito central

Esta aula trabalha o conceito de **Iframe e script revisitados** dentro do tópico **Elementos de mídia dinâmica**. O foco é consolidar o papel de iframe e script sem aprofundar JavaScript..

#### Objetivo da aula

Consolidar o papel de iframe e script sem aprofundar JavaScript.

#### Pré-requisitos

Conhecer mídia, recursos externos e segurança básica.

#### Resultado esperado

Reconhece iframe e script como pontos de integração com conteúdo externo e comportamento futuro.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
