# Aulas — 07.01 — Tabelas de dados
## Objetivo do tópico

Marcar dados tabulares com estrutura semântica e acessível.

## Aulas

### 07.01.01 — Estrutura básica de tabelas

#### Conceito central

Esta aula trabalha o conceito de **Estrutura básica de tabelas** dentro do tópico **Tabelas de dados**. O foco é usar table, tr, th e td para dados tabulares..

#### Objetivo da aula

Usar table, tr, th e td para dados tabulares.

#### Pré-requisitos

Conhecer estrutura semântica e listas.

#### Resultado esperado

Consegue criar uma tabela simples para dados reais.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

### 07.01.02 — Caption, scope e tabelas acessíveis

#### Conceito central

Esta aula trabalha o conceito de **Caption, scope e tabelas acessíveis** dentro do tópico **Tabelas de dados**. O foco é melhorar tabelas com caption e relações entre cabeçalhos e células..

#### Objetivo da aula

Melhorar tabelas com caption e relações entre cabeçalhos e células.

#### Pré-requisitos

Saber criar tabelas básicas.

#### Resultado esperado

Consegue tornar uma tabela mais compreensível e acessível.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
