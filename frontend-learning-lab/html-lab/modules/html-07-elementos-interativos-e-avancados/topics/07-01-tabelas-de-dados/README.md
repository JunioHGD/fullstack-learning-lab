# 07.01 — Tabelas de dados

## Módulo

HTML-07 — Elementos Interativos e Avançados

## Objetivo do tópico

Marcar dados tabulares com estrutura semântica e acessível.

## Aulas deste tópico

- **07.01.01 — Estrutura básica de tabelas**: Usar table, tr, th e td para dados tabulares.
- **07.01.02 — Caption, scope e tabelas acessíveis**: Melhorar tabelas com caption e relações entre cabeçalhos e células.
