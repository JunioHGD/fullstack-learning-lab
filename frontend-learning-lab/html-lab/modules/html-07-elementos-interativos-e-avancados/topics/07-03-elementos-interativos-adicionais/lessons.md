# Aulas — 07.03 — Elementos interativos adicionais
## Objetivo do tópico

Conhecer elementos nativos para revelação, diálogos, progresso e medição.

## Aulas

### 07.03.01 — Details e summary

#### Conceito central

Esta aula trabalha o conceito de **Details e summary** dentro do tópico **Elementos interativos adicionais**. O foco é criar blocos expansíveis com HTML nativo..

#### Objetivo da aula

Criar blocos expansíveis com HTML nativo.

#### Pré-requisitos

Conhecer elementos interativos nativos.

#### Resultado esperado

Consegue criar uma seção expansível sem JavaScript.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

### 07.03.02 — Dialog, progress e meter

#### Conceito central

Esta aula trabalha o conceito de **Dialog, progress e meter** dentro do tópico **Elementos interativos adicionais**. O foco é reconhecer usos semânticos de dialog, progress e meter..

#### Objetivo da aula

Reconhecer usos semânticos de dialog, progress e meter.

#### Pré-requisitos

Conhecer details e summary.

#### Resultado esperado

Diferencia diálogo, progresso e medição, sabendo seus limites em HTML puro.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
