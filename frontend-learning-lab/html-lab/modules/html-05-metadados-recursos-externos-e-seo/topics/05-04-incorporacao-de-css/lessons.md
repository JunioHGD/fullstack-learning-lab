# Aulas — 05.04 — Incorporação de CSS
## Objetivo do tópico

Conectar folhas de estilo ao HTML sem estudar CSS em profundidade.

## Aulas

### 05.04.01 — Link stylesheet e separação de responsabilidades

#### Conceito central

Esta aula trabalha o conceito de **Link stylesheet e separação de responsabilidades** dentro do tópico **Incorporação de CSS**. O foco é usar link rel stylesheet e explicar a separação entre estrutura e apresentação..

#### Objetivo da aula

Usar link rel stylesheet e explicar a separação entre estrutura e apresentação.

#### Pré-requisitos

Conhecer head e link.

#### Resultado esperado

Consegue preparar um HTML para receber estilos no CSS Lab.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão
