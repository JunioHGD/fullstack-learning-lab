# Aulas — 05.07 — Integração HTML, CSS, JavaScript e DOM
## Objetivo do tópico

Entender a conexão conceitual entre HTML e os Labs posteriores.

## Aulas

### 05.07.01 — HTML como base para CSS, JavaScript e DOM

#### Conceito central

Esta aula trabalha o conceito de **HTML como base para CSS, JavaScript e DOM** dentro do tópico **Integração HTML, CSS, JavaScript e DOM**. O foco é explicar como a marcação HTML será estilizada, manipulada e representada no DOM..

#### Objetivo da aula

Explicar como a marcação HTML será estilizada, manipulada e representada no DOM.

#### Pré-requisitos

Conhecer estrutura, recursos externos e semântica.

#### Resultado esperado

Consegue relacionar HTML aos próximos Labs sem misturar escopos.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Depois de conectar recursos, o aluno entende a separação de responsabilidades entre as tecnologias.
