# Aulas — 05.08 — Validação de HTML
## Objetivo do tópico

Validar marcação HTML e corrigir erros comuns.

## Aulas

### 05.08.01 — Validador HTML e correção de estrutura

#### Conceito central

Esta aula trabalha o conceito de **Validador HTML e correção de estrutura** dentro do tópico **Validação de HTML**. O foco é usar validação para identificar erros de sintaxe, aninhamento e atributos..

#### Objetivo da aula

Usar validação para identificar erros de sintaxe, aninhamento e atributos.

#### Pré-requisitos

Conhecer estrutura HTML completa.

#### Resultado esperado

Consegue validar uma página e registrar correções básicas.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
