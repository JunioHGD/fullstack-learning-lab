# Aulas — 05.01 — Metadados essenciais
## Objetivo do tópico

Configurar metadados fundamentais no head.

## Aulas

### 05.01.01 — Title, charset, viewport e description

#### Conceito central

Esta aula trabalha o conceito de **Title, charset, viewport e description** dentro do tópico **Metadados essenciais**. O foco é configurar metadados essenciais para páginas publicáveis..

#### Objetivo da aula

Configurar metadados essenciais para páginas publicáveis.

#### Pré-requisitos

Conhecer estrutura do head.

#### Resultado esperado

Consegue montar um head básico profissional.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Metadados essenciais vêm antes de ícones, redes sociais e recursos externos.
