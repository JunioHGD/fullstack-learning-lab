### 05.06.01 — Carregamento de CSS, scripts e DevTools introdutório

#### Conceito central

Esta aula trabalha o conceito de **Carregamento de CSS, scripts e DevTools introdutório** dentro do tópico **Carregamento de recursos e performance introdutória**. O foco é reconhecer impactos básicos de CSS, script, defer e async no carregamento..

#### Objetivo da aula

Reconhecer impactos básicos de CSS, script, defer e async no carregamento.

#### Pré-requisitos

Saber conectar CSS e JS.

#### Resultado esperado

Consegue inspecionar recursos básicos da página no navegador.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
