# Aulas — 05.05 — Incorporação de JavaScript
## Objetivo do tópico

Conectar scripts ao HTML sem estudar lógica JavaScript.

## Aulas

### 05.05.01 — Script, defer, async e type module

#### Conceito central

Esta aula trabalha o conceito de **Script, defer, async e type module** dentro do tópico **Incorporação de JavaScript**. O foco é inserir scripts externos com atenção ao carregamento..

#### Objetivo da aula

Inserir scripts externos com atenção ao carregamento.

#### Pré-requisitos

Conhecer estrutura do documento e recursos externos.

#### Resultado esperado

Consegue conectar um script externo de forma adequada.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

JavaScript entra após CSS porque adiciona comportamento à estrutura já marcada e estilizada futuramente.
