# HTML-05 — Metadados, Recursos Externos e SEO

## Objetivo do módulo

Configurar documentos HTML para publicação, recursos externos, SEO básico, integração CSS/JS e validação.

## Pré-requisitos

HTML-00 a HTML-04.

## Tópicos do módulo

- **05.01 — Metadados essenciais**: Configurar metadados fundamentais no head.
- **05.02 — Favicon e ícones**: Adicionar ícones e entender links de recurso.
- **05.03 — Open Graph e mídias sociais**: Conhecer metadados básicos para prévias sociais.
- **05.04 — Incorporação de CSS**: Conectar folhas de estilo ao HTML sem estudar CSS em profundidade.
- **05.05 — Incorporação de JavaScript**: Conectar scripts ao HTML sem estudar lógica JavaScript.
- **05.06 — Carregamento de recursos e performance introdutória**: Entender noções básicas de carregamento de CSS e JS.
- **05.07 — Integração HTML, CSS, JavaScript e DOM**: Entender a conexão conceitual entre HTML e os Labs posteriores.
- **05.08 — Validação de HTML**: Validar marcação HTML e corrigir erros comuns.

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
