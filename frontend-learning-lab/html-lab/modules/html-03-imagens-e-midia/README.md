# HTML-03 — Imagens e Mídia

## Objetivo do módulo

Aprofundar o uso de imagens, figuras, áudio, vídeo, iframes e templates em HTML.

## Pré-requisitos

HTML-01 e HTML-02.

## Tópicos do módulo

- **03.01 — Imagens responsivas**: Fornecer imagens adequadas para diferentes contextos de tela e resolução.
- **03.02 — Figure e figcaption**: Associar imagens e legendas de forma semântica.
- **03.03 — SVG e gráficos**: Conhecer formas de inserir SVG e reconhecer o limite do canvas no HTML Lab.
- **03.04 — Áudio e vídeo**: Incorporar mídia nativa com controles e fontes alternativas.
- **03.05 — Conteúdos incorporados**: Incorporar documentos e recursos externos usando iframe.
- **03.06 — Templates**: Conhecer template como HTML inativo preparado para uso futuro com JavaScript.

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
