# Aulas — 03.03 — SVG e gráficos
## Objetivo do tópico

Conhecer formas de inserir SVG e reconhecer o limite do canvas no HTML Lab.

## Aulas

### 03.03.01 — SVG via img, SVG inline e canvas introdutório

#### Conceito central

Esta aula trabalha o conceito de **SVG via img, SVG inline e canvas introdutório** dentro do tópico **SVG e gráficos**. O foco é comparar uso de SVG externo, SVG inline e canvas em nível conceitual..

#### Objetivo da aula

Comparar uso de SVG externo, SVG inline e canvas em nível conceitual.

#### Pré-requisitos

Conhecer imagens e figuras.

#### Resultado esperado

Entende quando usar SVG e que canvas depende de JavaScript para desenho.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
