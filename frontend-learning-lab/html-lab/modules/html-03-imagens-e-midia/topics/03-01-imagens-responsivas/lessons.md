# Aulas — 03.01 — Imagens responsivas
## Objetivo do tópico

Fornecer imagens adequadas para diferentes contextos de tela e resolução.

## Aulas

### 03.01.01 — Srcset, sizes e densidade de imagem

#### Conceito central

Esta aula trabalha o conceito de **Srcset, sizes e densidade de imagem** dentro do tópico **Imagens responsivas**. O foco é usar srcset e sizes para melhorar entrega de imagens..

#### Objetivo da aula

Usar srcset e sizes para melhorar entrega de imagens.

#### Pré-requisitos

Saber usar img, src e alt.

#### Resultado esperado

Reconhece como o navegador escolhe imagens alternativas.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

### 03.01.02 — Picture, source e art direction

#### Conceito central

#### Pré-requisitos

#### Resultado esperado
