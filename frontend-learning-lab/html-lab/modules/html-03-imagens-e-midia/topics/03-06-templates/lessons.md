# Aulas — 03.06 — Templates
## Objetivo do tópico

Conhecer template como HTML inativo preparado para uso futuro com JavaScript.

## Aulas

### 03.06.01 — Template como marcação inativa

#### Conceito central

Esta aula trabalha o conceito de **Template como marcação inativa** dentro do tópico **Templates**. O foco é reconhecer template como bloco de HTML não renderizado imediatamente..

#### Objetivo da aula

Reconhecer template como bloco de HTML não renderizado imediatamente.

#### Pré-requisitos

Conhecer estrutura HTML e elementos de conteúdo.

#### Resultado esperado

Entende que template prepara marcação reutilizável para estudos futuros de DOM/JS.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Fecha o módulo com uma ponte para DOM e JavaScript sem sair do escopo de HTML.
