# Aulas — 02.01 — Divisores genéricos
## Objetivo do tópico

Entender div e span como elementos genéricos sem significado específico.

## Aulas

### 02.01.01 — Div e span no agrupamento de conteúdo

#### Conceito central

Esta aula trabalha o conceito de **Div e span no agrupamento de conteúdo** dentro do tópico **Divisores genéricos**. O foco é usar div e span apenas quando não houver elemento semântico mais apropriado..

#### Objetivo da aula

Usar div e span apenas quando não houver elemento semântico mais apropriado.

#### Pré-requisitos

Conhecer elementos e atributos básicos.

#### Resultado esperado

Reconhece o limite semântico de div e span.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Começa pelos elementos genéricos para contrastá-los com opções semânticas posteriores.
