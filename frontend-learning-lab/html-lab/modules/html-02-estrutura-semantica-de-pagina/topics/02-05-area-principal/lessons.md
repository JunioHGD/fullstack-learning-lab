# Aulas — 02.05 — Área principal
## Objetivo do tópico

Identificar e marcar o conteúdo principal único da página.

## Aulas

### 02.05.01 — Main e conteúdo principal

#### Conceito central

Esta aula trabalha o conceito de **Main e conteúdo principal** dentro do tópico **Área principal**. O foco é usar main para delimitar o conteúdo principal da página..

#### Objetivo da aula

Usar main para delimitar o conteúdo principal da página.

#### Pré-requisitos

Conhecer header, footer e nav.

#### Resultado esperado

Consegue identificar o conteúdo principal e evitar múltiplos main indevidos.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Main vem após header/nav porque representa a região central do documento.
