# Aulas — 02.07 — Landmarks e acessibilidade estrutural
## Objetivo do tópico

Consolidar regiões semânticas como landmarks acessíveis.

## Aulas

### 02.07.01 — Landmarks semânticas de página

#### Conceito central

Esta aula trabalha o conceito de **Landmarks semânticas de página** dentro do tópico **Landmarks e acessibilidade estrutural**. O foco é reconhecer header, nav, main, aside e footer como regiões estruturais relevantes..

#### Objetivo da aula

Reconhecer header, nav, main, aside e footer como regiões estruturais relevantes.

#### Pré-requisitos

Conhecer elementos semânticos de página.

#### Resultado esperado

Consegue montar uma página semântica completa e navegável por regiões.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Fecha o módulo reunindo todos os elementos estruturais em uma página completa.
