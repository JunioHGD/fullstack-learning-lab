# Aulas — 02.04 — Navegação
## Objetivo do tópico

Marcar áreas de navegação principal e secundária.

## Aulas

### 02.04.01 — Nav e menus de navegação

#### Conceito central

Esta aula trabalha o conceito de **Nav e menus de navegação** dentro do tópico **Navegação**. O foco é usar nav para grupos importantes de links de navegação..

#### Objetivo da aula

Usar nav para grupos importantes de links de navegação.

#### Pré-requisitos

Saber criar links internos e externos.

#### Resultado esperado

Consegue estruturar uma navegação semântica básica.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

A navegação depende do domínio prévio de links e da estrutura global da página.
