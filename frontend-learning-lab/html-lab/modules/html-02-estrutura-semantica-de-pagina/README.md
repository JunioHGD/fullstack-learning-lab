# HTML-02 — Estrutura Semântica de Página

## Objetivo do módulo

Ensinar a organizar páginas em regiões semânticas com elementos estruturais e landmarks.

## Pré-requisitos

HTML-01 - Conteúdo de Texto e Hiperlinks.

## Tópicos do módulo

- **02.01 — Divisores genéricos**: Entender div e span como elementos genéricos sem significado específico.
- **02.02 — Section e article**: Dividir conteúdo em seções temáticas e unidades independentes.
- **02.03 — Header e footer**: Marcar cabeçalhos e rodapés de páginas ou seções.
- **02.04 — Navegação**: Marcar áreas de navegação principal e secundária.
- **02.05 — Área principal**: Identificar e marcar o conteúdo principal único da página.
- **02.06 — Aside e conteúdo complementar**: Representar conteúdo complementar ao conteúdo principal.
- **02.07 — Landmarks e acessibilidade estrutural**: Consolidar regiões semânticas como landmarks acessíveis.

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada tópico em `topics/`.
- Em cada tópico, estude `lessons.md`, responda `assimilation-questions.md` e pratique em `exercises/`.
