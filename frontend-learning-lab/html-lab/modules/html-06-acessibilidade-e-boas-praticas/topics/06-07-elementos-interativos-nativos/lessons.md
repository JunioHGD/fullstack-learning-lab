# Aulas — 06.07 — Elementos interativos nativos
## Objetivo do tópico

Priorizar elementos HTML nativos para interações acessíveis.

## Aulas

### 06.07.01 — Button, link, details, progress e meter

#### Conceito central

Esta aula trabalha o conceito de **Button, link, details, progress e meter** dentro do tópico **Elementos interativos nativos**. O foco é reconhecer controles nativos e seus benefícios de acessibilidade..

#### Objetivo da aula

Reconhecer controles nativos e seus benefícios de acessibilidade.

#### Pré-requisitos

Conhecer links, botões e noções de foco.

#### Resultado esperado

Evita criar interatividade com div ou span quando existe elemento nativo.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Surge após foco, teclado e ARIA para mostrar que controles nativos reduzem complexidade.
