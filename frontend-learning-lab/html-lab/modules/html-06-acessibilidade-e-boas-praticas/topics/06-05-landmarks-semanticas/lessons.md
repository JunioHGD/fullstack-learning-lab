# Aulas — 06.05 — Landmarks semânticas
## Objetivo do tópico

Reforçar regiões de página como mecanismos estruturais de acessibilidade.

## Aulas

### 06.05.01 — Landmarks implícitas e estrutura acessível

#### Conceito central

Esta aula trabalha o conceito de **Landmarks implícitas e estrutura acessível** dentro do tópico **Landmarks semânticas**. O foco é revisar header, nav, main, aside e footer como landmarks implícitas..

#### Objetivo da aula

Revisar header, nav, main, aside e footer como landmarks implícitas.

#### Pré-requisitos

Conhecer estrutura semântica de página.

#### Resultado esperado

Consegue revisar se uma página possui regiões claras e coerentes.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.
