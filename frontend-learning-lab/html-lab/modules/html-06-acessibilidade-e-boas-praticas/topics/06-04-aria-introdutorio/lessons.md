# Aulas — 06.04 — ARIA introdutório
## Objetivo do tópico

Entender ARIA como complemento, não substituto, de HTML semântico.

## Aulas

### 06.04.01 — Role, aria-label e aria-describedby

#### Conceito central

Esta aula trabalha o conceito de **Role, aria-label e aria-describedby** dentro do tópico **ARIA introdutório**. O foco é reconhecer usos básicos de ARIA e a regra de priorizar HTML nativo..

#### Objetivo da aula

Reconhecer usos básicos de ARIA e a regra de priorizar HTML nativo.

#### Pré-requisitos

Conhecer semântica e formulários acessíveis.

#### Resultado esperado

Evita ARIA redundante quando um elemento HTML resolve o problema.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

ARIA é apresentado após semântica e foco para evitar uso prematuro ou redundante.
