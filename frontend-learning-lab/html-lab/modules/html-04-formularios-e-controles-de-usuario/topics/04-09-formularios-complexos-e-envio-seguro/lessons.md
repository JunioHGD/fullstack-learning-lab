# Aulas — 04.09 — Formulários complexos e envio seguro
## Objetivo do tópico

Reconhecer cuidados básicos de envio e limites entre HTML e backend.

## Aulas

### 04.09.01 — Cuidados básicos no envio de dados

#### Conceito central

Esta aula trabalha o conceito de **Cuidados básicos no envio de dados** dentro do tópico **Formulários complexos e envio seguro**. O foco é entender limites de GET/POST, exposição de dados em URL e necessidade de validação no servidor..

#### Objetivo da aula

Entender limites de GET/POST, exposição de dados em URL e necessidade de validação no servidor.

#### Pré-requisitos

Conhecer formulários e validação nativa.

#### Resultado esperado

Sabe que HTML não substitui segurança e validação backend.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão
