# 04.06 — Validação HTML

## Módulo

HTML-04 — Formulários e Controles de Usuário

## Objetivo do tópico

Aplicar validação nativa de formulários com atributos HTML.

## Aulas deste tópico

- **04.06.01 — Required e validação por tipo**: Usar required e tipos como email, url e number para validação nativa.
- **04.06.02 — Min, max, step, minlength, maxlength e pattern**: Aplicar restrições de tamanho, intervalo, passo e padrão.

- `assimilation-questions.md`: perguntas para fixação.
- `exercises/`: exercícios práticos, incluindo código quando necessário.
