# Aulas — 04.06 — Validação HTML
## Objetivo do tópico

Aplicar validação nativa de formulários com atributos HTML.

## Aulas

### 04.06.01 — Required e validação por tipo

#### Conceito central

Esta aula trabalha o conceito de **Required e validação por tipo** dentro do tópico **Validação HTML**. O foco é usar required e tipos como email, url e number para validação nativa..

#### Objetivo da aula

Usar required e tipos como email, url e number para validação nativa.

#### Pré-requisitos

Conhecer tipos de input.

#### Resultado esperado

Consegue criar validações nativas básicas.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

### 04.06.02 — Min, max, step, minlength, maxlength e pattern

#### Conceito central

Esta aula trabalha o conceito de **Min, max, step, minlength, maxlength e pattern** dentro do tópico **Validação HTML**. O foco é aplicar restrições de tamanho, intervalo, passo e padrão..

#### Objetivo da aula

Aplicar restrições de tamanho, intervalo, passo e padrão.

#### Pré-requisitos

Saber usar required e validação por tipo.

#### Resultado esperado

Consegue aplicar regras de validação nativa mais específicas.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão
