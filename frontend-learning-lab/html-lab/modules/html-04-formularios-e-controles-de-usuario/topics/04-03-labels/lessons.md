# Aulas — 04.03 — Labels
## Objetivo do tópico

Associar rótulos a controles para usabilidade e acessibilidade.

## Aulas

### 04.03.01 — Label, for, id e associação explícita

#### Conceito central

Esta aula trabalha o conceito de **Label, for, id e associação explícita** dentro do tópico **Labels**. O foco é vincular labels a campos usando for e id ou associação implícita..

#### Objetivo da aula

Vincular labels a campos usando for e id ou associação implícita.

#### Pré-requisitos

Conhecer inputs comuns.

#### Resultado esperado

Consegue garantir que campos tenham rótulos claros e clicáveis.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Labels aparecem após inputs porque dependem da existência de controles a serem rotulados.
