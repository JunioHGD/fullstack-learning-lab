# Aulas — 04.02 — Inputs comuns
## Objetivo do tópico

Conhecer os principais tipos de input e atributos essenciais.

## Aulas

### 04.02.01 — Inputs textuais e atributos name, value e placeholder

#### Conceito central

Esta aula trabalha o conceito de **Inputs textuais e atributos name, value e placeholder** dentro do tópico **Inputs comuns**. O foco é usar inputs textuais e atributos fundamentais de envio..

#### Objetivo da aula

Usar inputs textuais e atributos fundamentais de envio.

#### Pré-requisitos

Conhecer estrutura de form.

#### Resultado esperado

Consegue criar campos textuais com name, value e placeholder.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

### 04.02.02 — Inputs especializados: email, url, number, range, date e color

#### Conceito central

Esta aula trabalha o conceito de **Inputs especializados: email, url, number, range, date e color** dentro do tópico **Inputs comuns**. O foco é escolher tipos de input adequados ao tipo de dado..

#### Objetivo da aula

Escolher tipos de input adequados ao tipo de dado.

#### Pré-requisitos

Saber criar inputs textuais.

#### Resultado esperado

Reconhece inputs com validações e comportamentos nativos.

#### Como estudar

1. Leia o conceito central com atenção.
