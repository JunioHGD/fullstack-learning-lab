# HTML-04 — Formulários e Controles de Usuário

## Objetivo do módulo

Criar formulários HTML completos, acessíveis, validáveis e preparados para integração futura com backend.

## Pré-requisitos

HTML-00 a HTML-03, especialmente links, botões, atributos e semântica básica.

## Tópicos do módulo

- **04.01 — Formulário básico**: Entender a estrutura e o propósito de um formulário HTML.
- **04.02 — Inputs comuns**: Conhecer os principais tipos de input e atributos essenciais.
- **04.03 — Labels**: Associar rótulos a controles para usabilidade e acessibilidade.
- **04.04 — Grupos de campos**: Agrupar controles relacionados com semântica adequada.
- **04.05 — Select e opções**: Criar controles de escolha, grupos de opções e sugestões.
- **04.06 — Validação HTML**: Aplicar validação nativa de formulários com atributos HTML.
- **04.07 — Acessibilidade em formulários**: Aplicar práticas básicas de acessibilidade em formulários.
- **04.08 — Botões e interatividade**: Diferenciar botões de envio, reset e ação neutra.
- **04.09 — Formulários complexos e envio seguro**: Reconhecer cuidados básicos de envio e limites entre HTML e backend.
