# Aulas — 01.06 — Imagens simples
## Objetivo do tópico

Inserir imagens simples com texto alternativo básico.

## Aulas

### 01.06.01 — Img, src, alt, width e height

#### Conceito central

Esta aula trabalha o conceito de **Img, src, alt, width e height** dentro do tópico **Imagens simples**. O foco é inserir imagens com src, alt e dimensões básicas..

#### Objetivo da aula

Inserir imagens com src, alt e dimensões básicas.

#### Pré-requisitos

Conhecer atributos e caminhos básicos.

#### Resultado esperado

Consegue adicionar imagens simples com texto alternativo adequado.

#### Como estudar

1. Leia o conceito central com atenção.
2. Reescreva a ideia principal com suas próprias palavras.
3. Relacione o conceito com uma página HTML real.
4. Responda as perguntas de assimilação.
5. Crie pelo menos um exercício ou exemplo prático em `exercises/`.

## Observações de progressão

Imagens entram após links porque também dependem de caminhos e atributos.
