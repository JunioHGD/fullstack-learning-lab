# 01.05 — Links e âncoras

## Módulo

HTML-01 — Conteúdo de Texto e Hiperlinks

## Objetivo do tópico

Criar navegação entre páginas, seções e recursos externos.

## Aulas deste tópico

- **01.05.01 — Links internos, externos e de seção**: Criar links com href para páginas, URLs externas e âncoras internas.
- **01.05.02 — Links seguros e texto descritivo**: Aplicar texto de link claro e atributos target e rel quando necessário.
