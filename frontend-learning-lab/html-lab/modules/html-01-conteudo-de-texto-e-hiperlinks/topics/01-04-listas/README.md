# 01.04 — Listas

## Módulo

HTML-01 — Conteúdo de Texto e Hiperlinks

## Objetivo do tópico

Representar sequências, coleções e descrições usando listas adequadas.

## Aulas deste tópico

- **01.04.01 — Listas ordenadas e não ordenadas**: Usar ol, ul e li para representar sequências e coleções.
- **01.04.02 — Listas de descrição e listas aninhadas**: Usar dl, dt, dd e listas aninhadas em estruturas mais ricas.
