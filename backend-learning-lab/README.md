# Backend Learning Lab

## Objetivo

Organizar a trilha completa de backend moderno com JavaScript/TypeScript e Node.js, cobrindo fundamentos web, HTTP, APIs, validação, banco relacional, PostgreSQL, autenticação, segurança, testes, observabilidade, Docker, deploy e CI/CD.

## Papel na formação

O Backend Learning Lab funciona como a base técnica responsável por transformar fundamentos web e JavaScript em capacidade profissional de construir, manter, proteger, testar e entregar APIs reais em uma formação fullstack.

## Pré-requisitos

- Lógica de programação
- JavaScript básico e moderno
- Funções, objetos, arrays e módulos
- Promises, async/await e noções de assincronicidade
- Noções gerais de web, navegador e aplicações cliente-servidor

## Estrutura do lab

Este lab usa uma estrutura compacta baseada em tópicos:

```txt
backend-learning-lab/
├── README.md
├── docs/
│   ├── index.html
│   ├── assets/
│   │   ├── css/
│   │   ├── js/
│   │   └── images/
│   └── projects/
└── modules/
```

## Como estudar

1. Estude os módulos em ordem.
2. Use o `README.md` de cada módulo para entender o escopo.
3. Use o `checklist.md` para controlar tópicos, aulas e prática.
4. Use `lessons.md` dentro de cada tópico para organizar os conceitos.
5. Use `assimilation-questions.md` para revisar entendimento.
6. Use `exercises/` para guardar exercícios e código.
7. Ao concluir um módulo, desenvolva o projeto correspondente em `docs/projects/`.
