# BACKEND-12 — Ambientes, Docker, Deploy e CI/CD

## Objetivo

Fechar a trilha com ambientes, configuração, build, Docker, Compose, deploy, CI/CD, migrations e segredos no pipeline.

## Pré-requisitos

BACKEND-11.

## Progressão didática

Define ambientes e configuração, prepara build, containeriza, compõe serviços locais e automatiza testes, deploy, migrations e segredos.
