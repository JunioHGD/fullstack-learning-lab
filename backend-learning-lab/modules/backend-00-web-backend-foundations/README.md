# BACKEND-00 — Fundamentos da Web e do Backend

## Objetivo

Estabelecer a visão sistêmica do backend em aplicações web e fullstack, diferenciando cliente, servidor, regras de negócio, recursos, estado e integração.

## Pré-requisitos

Lógica de programação; JavaScript básico; noções gerais de aplicações web.

## Progressão didática

Começa pela função do backend, avança para arquitetura cliente-servidor e fecha com recursos, estado e regras de negócio antes de qualquer implementação.
