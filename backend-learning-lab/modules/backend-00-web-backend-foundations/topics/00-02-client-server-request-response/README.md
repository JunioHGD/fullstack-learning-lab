# 00.02 — Arquitetura cliente-servidor e ciclo request-response

## Módulo

BACKEND-00 — Fundamentos da Web e do Backend

## Objetivo do tópico

Compreender como clientes e servidores se comunicam e como o ciclo de requisição e resposta organiza a web.

## Aulas do tópico

| Código | Aula | Objetivo | Resultado esperado |
|---|---|---|---|
| 00.02.01 | Arquitetura cliente-servidor | Diferenciar cliente, servidor e responsabilidades de cada lado da aplicação. | Identificar o papel do cliente e do servidor em um fluxo web. |
| 00.02.02 | Ciclo de requisição e resposta | Entender o caminho básico de uma requisição do cliente até a resposta do servidor. | Explicar o fluxo request-response em alto nível. |
