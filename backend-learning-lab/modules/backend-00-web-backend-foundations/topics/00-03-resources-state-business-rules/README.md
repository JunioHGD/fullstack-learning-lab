# 00.03 — Recursos, estado e regras de negócio

## Módulo

BACKEND-00 — Fundamentos da Web e do Backend

## Objetivo do tópico

Introduzir recursos, estado e regras de negócio como responsabilidades centrais do backend.

## Aulas do tópico

| Código | Aula | Objetivo | Resultado esperado |
|---|---|---|---|
| 00.03.01 | Recursos e regras de negócio | Entender como recursos representam entidades manipuladas pela aplicação e como regras controlam operações. | Diferenciar recurso, dado e regra de negócio. |
| 00.03.02 | Processamento stateless e stateful | Compreender diferenças iniciais entre processamento sem estado e com estado. | Reconhecer quando uma interação depende ou não de estado mantido. |
| 00.03.03 | Integração backend-frontend em alto nível | Relacionar backend e frontend sem aprofundar implementação de interface. | Explicar como frontend consome dados e ações fornecidas pelo backend. |

## Arquivos deste tópico

- `lessons.md`: explicação organizada dos conceitos do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios, códigos, experimentos e práticas.
