# Lessons — 00.01 — Papel do backend em aplicações web

## Objetivo do tópico

Delimitar o que é backend, quais responsabilidades ele assume e como se diferencia do frontend.

---

## 00.01.01 — O que é backend em uma aplicação web

### Objetivo

Entender o backend como camada responsável por receber requisições, aplicar regras e devolver respostas.

### Pré-requisitos

Lógica de programação; noções gerais de web.

### Resultado esperado

Explicar o que é backend e quais problemas ele resolve.

### Conceitos para estudar

- Entenda o conceito central de **O que é backend em uma aplicação web**.
- Relacione o conceito com o fluxo de uma aplicação backend real.
- Identifique onde esse conhecimento aparece em APIs, serviços, banco de dados, segurança, testes ou deploy.
- Registre exemplos curtos e dúvidas no próprio processo de estudo.

### Observações

Use esta seção como base de estudo. A explicação detalhada, exemplos de código e anotações pessoais podem ser adicionados gradualmente conforme o tópico for estudado.
