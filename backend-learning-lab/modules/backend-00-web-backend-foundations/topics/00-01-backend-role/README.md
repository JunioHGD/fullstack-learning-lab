# 00.01 — Papel do backend em aplicações web

## Módulo

BACKEND-00 — Fundamentos da Web e do Backend

## Objetivo do tópico

Delimitar o que é backend, quais responsabilidades ele assume e como se diferencia do frontend.

## Aulas do tópico

| Código | Aula | Objetivo | Resultado esperado |
|---|---|---|---|
| 00.01.01 | O que é backend em uma aplicação web | Entender o backend como camada responsável por receber requisições, aplicar regras e devolver respostas. | Explicar o que é backend e quais problemas ele resolve. |
| 00.01.02 | Papel do backend na arquitetura fullstack | Relacionar backend, frontend, dados e regras de negócio dentro de uma aplicação fullstack. | Descrever como backend sustenta uma aplicação fullstack. |

## Arquivos deste tópico

- `lessons.md`: explicação organizada dos conceitos do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios, códigos, experimentos e práticas.
