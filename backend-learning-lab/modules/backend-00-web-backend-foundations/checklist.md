# Checklist — BACKEND-00 — Fundamentos da Web e do Backend

## Objetivo do módulo

Estabelecer a visão sistêmica do backend em aplicações web e fullstack, diferenciando cliente, servidor, regras de negócio, recursos, estado e integração.

## Como usar este checklist

Marque cada item conforme avançar. O checklist representa os conteúdos do módulo de forma organizada por tópico e por aula.

## 1. 00.01 — Papel do backend em aplicações web

**Objetivo do tópico:** Delimitar o que é backend, quais responsabilidades ele assume e como se diferencia do frontend.
