# BACKEND-04 — APIs com Express e Organização em Camadas

## Objetivo

Construir APIs HTTP com framework e organizar o fluxo em rotas, middlewares, controllers, services e repositories.

## Pré-requisitos

BACKEND-03.

## Progressão didática

Inicia com frameworks, depois rotas e middlewares, e só então separa responsabilidades em camadas.
