# 02.04 — Servidor HTTP sem framework

## Módulo

BACKEND-02 — Node.js e Modelo de Execução Backend

## Objetivo do tópico

Criar a base conceitual de servidores HTTP antes de usar frameworks.

## Aulas do tópico

| Código | Aula | Objetivo | Resultado esperado |
|---|---|---|---|
| 02.04.01 | Servidor HTTP com node:http | Entender a criação de servidor HTTP usando apenas módulo nativo. | Explicar request, response e resposta JSON em servidor mínimo. |
| 02.04.02 | Limites do servidor sem framework | Reconhecer problemas que frameworks resolvem. | Justificar o uso posterior de Express ou Fastify. |

## Arquivos deste tópico

- `lessons.md`: explicação organizada dos conceitos do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios, códigos, experimentos e práticas.
