# BACKEND-02 — Node.js e Modelo de Execução Backend

## Objetivo

Introduzir Node.js como runtime backend e explicar módulos, event loop, I/O assíncrono e servidor HTTP sem framework.

## Pré-requisitos

BACKEND-01; JavaScript assíncrono básico.

## Progressão didática

Começa pela identidade do runtime, avança para módulos nativos e modelo assíncrono, e fecha com servidor HTTP sem framework.
