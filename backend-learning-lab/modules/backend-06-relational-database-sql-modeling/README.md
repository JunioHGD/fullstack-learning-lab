# BACKEND-06 — Banco Relacional, SQL e Modelagem de Dados

## Objetivo

Formar base de persistência relacional com modelagem, integridade, SQL, relacionamentos, constraints e índices.

## Pré-requisitos

BACKEND-05.

## Progressão didática

Começa pelo modelo relacional, depois integridade e relacionamentos, e só então consultas e otimização inicial.
