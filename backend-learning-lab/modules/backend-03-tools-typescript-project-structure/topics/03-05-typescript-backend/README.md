# 03.05 — TypeScript aplicado ao backend

## Módulo

BACKEND-03 — Ferramentas, TypeScript e Estrutura de Projeto

## Objetivo do tópico

Usar TypeScript de forma aplicada a contratos internos backend.

## Aulas do tópico

| Código | Aula | Objetivo | Resultado esperado |
|---|---|---|---|
| 03.05.01 | tsconfig.json para Node.js | Entender configuração TypeScript essencial para backend Node.js. | Interpretar configurações iniciais do tsconfig.json. |
| 03.05.02 | DTOs e contratos internos tipados | Aplicar tipagem em entradas, saídas e objetos de transferência. | Reconhecer como DTOs ajudam a estruturar contratos internos. |

## Arquivos deste tópico

- `lessons.md`: explicação organizada dos conceitos do tópico.
- `assimilation-questions.md`: perguntas para fixação e revisão ativa.
- `exercises/`: espaço para exercícios, códigos, experimentos e práticas.
