# BACKEND-03 — Ferramentas, TypeScript e Estrutura de Projeto

## Objetivo

Organizar ambiente profissional com npm, package.json, dependências, módulos, variáveis de ambiente, TypeScript e estrutura de pastas.

## Pré-requisitos

BACKEND-02; JavaScript moderno.

## Progressão didática

Estrutura o projeto e dependências, depois module system e configuração, e por fim TypeScript e organização interna.

## Tópicos do módulo

| Código | Tópico | Pasta | Aulas |
|---|---|---|---:|
| 03.01 | npm, criação de projeto e package.json | `03-01-npm-package-json` | 2 |
| 03.02 | Dependências e lockfile | `03-02-dependencies-lockfile` | 2 |
| 03.03 | CommonJS e ESM | `03-03-commonjs-esm` | 2 |
| 03.04 | Variáveis de ambiente e arquivos .env | `03-04-environment-configuration` | 2 |
| 03.05 | TypeScript aplicado ao backend | `03-05-typescript-backend` | 2 |
| 03.06 | Estrutura de pastas e organização por responsabilidade | `03-06-project-structure-responsibility` | 2 |

## Arquivos do módulo

- `checklist.md`: controle de progresso do módulo.
- `interview-questions.md`: perguntas integradas para revisão e preparação para entrevista.
- `topics/`: tópicos organizados em sequência, cada um com aulas, perguntas e exercícios.
