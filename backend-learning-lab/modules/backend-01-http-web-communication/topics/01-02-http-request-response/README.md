# 01.02 — Requisição e resposta HTTP

## Módulo

BACKEND-01 — HTTP e Comunicação Web

## Objetivo do tópico

Analisar a estrutura básica de requests e responses.

## Aulas do tópico

| Código | Aula | Objetivo | Resultado esperado |
|---|---|---|---|
| 01.02.01 | Requisição HTTP | Entender a composição de uma requisição enviada por um cliente. | Identificar método, URL, headers e body de uma requisição. |
| 01.02.02 | Resposta HTTP | Entender a composição de uma resposta gerada pelo servidor. | Identificar status, headers e body de uma resposta. |
