# BACKEND-01 — HTTP e Comunicação Web

## Objetivo

Construir a base protocolar necessária para APIs corretas, previsíveis e integráveis com clientes web.

## Pré-requisitos

BACKEND-00.

## Progressão didática

Parte do protocolo HTTP, passa pela anatomia de mensagens, métodos e status codes, e conclui com headers, JSON, cookies e CORS.

## Tópicos do módulo

| Código | Tópico | Pasta | Aulas |

| 01.01 | Fundamentos do protocolo HTTP | `01-01-http-foundations` | 2 |
| 01.02 | Requisição e resposta HTTP | `01-02-http-request-response` | 2 |
| 01.03 | Métodos HTTP e propriedades | `01-03-http-methods-properties` | 2 |
