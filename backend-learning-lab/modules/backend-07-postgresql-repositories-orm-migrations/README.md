# BACKEND-07 — PostgreSQL, Repositórios, ORM e Migrations

## Objetivo

Aplicar persistência real com PostgreSQL, transações, repositories, ORM e migrations.

## Pré-requisitos

BACKEND-06.

## Progressão didática

Começa no banco concreto, passa por transações e acesso direto a dados, depois introduz ORM e migrations.
