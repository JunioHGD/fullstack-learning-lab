# BACKEND-08 — Autenticação, Autorização e Sessões

## Objetivo

Estruturar identidade, estado autenticado, sessões, tokens, JWT, senhas e autorização.

## Pré-requisitos

BACKEND-05; BACKEND-07.

## Progressão didática

Separa autenticação de autorização, introduz mecanismos de estado e credenciais, e conclui com proteção de recursos.
