# Programming Foundations Lab

O **Programming Foundations Lab** organiza o estudo dos fundamentos de programação, lógica de programação e pensamento computacional de forma progressiva, sem depender inicialmente de uma tecnologia específica.

## Objetivo

Construir uma base sólida para aprender JavaScript, DOM, backend, APIs, bancos de dados, testes, arquitetura e projetos fullstack com mais segurança conceitual.

## Como este lab está organizado

- `modules/`: módulos de estudo organizados em sequência.
- `topics/`: tópicos de cada módulo.
- `lessons.md`: explicação dos conceitos de cada tópico.
- `assimilation-questions.md`: perguntas de fixação por tópico.
- `exercises/`: espaço para exercícios práticos, pseudocódigo e códigos simples.
- `docs/projects/`: projetos de conclusão de módulo.

## Como estudar

1. Siga os módulos em ordem.
2. Leia o `README.md` do módulo para entender o escopo.
3. Use o `checklist.md` como mapa de progresso.
4. Estude cada tópico pelo arquivo `lessons.md`.
5. Responda às perguntas em `assimilation-questions.md`.
6. Crie exercícios dentro de `exercises/` conforme avançar.
7. Ao finalizar um módulo, desenvolva um projeto de conclusão em `docs/projects/`.

## Observação
