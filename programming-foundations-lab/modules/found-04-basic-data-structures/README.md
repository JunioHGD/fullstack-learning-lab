# FOUND-04 — Estruturas de Dados Básicas: Vetores, Objetos e Strings

## Objetivo do módulo

Explorar estruturas fundamentais para armazenar coleções, textos e dados relacionados.

## Pré-requisitos

Variáveis, controle de fluxo, funções, escopo e noções iniciais de mutabilidade/referência.

## Tópicos do módulo

- `found-04-01-arrays` — FOUND-04.01 — Vetores e arrays
- `found-04-02-array-operations` — FOUND-04.02 — Operações básicas em vetores
- `found-04-03-strings` — FOUND-04.03 — Strings e manipulação básica de texto
- `found-04-04-objects-records` — FOUND-04.04 — Objetos e registros
- `found-04-05-maps-dictionaries` — FOUND-04.05 — Mapas e dicionários em nível conceitual

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-04-basic-data-structures/`.
