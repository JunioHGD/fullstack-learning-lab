# Aulas — FOUND-03.02 — Escopo de variáveis

## Objetivo do tópico

Compreender onde variáveis existem e de onde podem ser acessadas.

## Conceitos do tópico

### FOUND-03.02.01 — Escopo local e global

#### Objetivo

Diferenciar variáveis locais e globais em programas simples.

#### Pré-requisitos

Funções e variáveis.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Escopo de variáveis**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Escopo local e global** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Prever se uma variável pode ser acessada em determinado ponto do código.
