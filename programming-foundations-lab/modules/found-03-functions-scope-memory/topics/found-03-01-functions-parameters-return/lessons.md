# Aulas — FOUND-03.01 — Definição de funções, parâmetros, argumentos e retorno

## Objetivo do tópico

Entender funções como blocos reutilizáveis que recebem entradas e podem devolver resultados.

## Conceitos do tópico

### FOUND-03.01.01 — O que são funções

#### Objetivo

Compreender função como bloco nomeado e reutilizável de lógica.

#### Pré-requisitos

Controle de fluxo e variáveis.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Definição de funções, parâmetros, argumentos e retorno**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Relacione este conceito com os tópicos anteriores do módulo.
