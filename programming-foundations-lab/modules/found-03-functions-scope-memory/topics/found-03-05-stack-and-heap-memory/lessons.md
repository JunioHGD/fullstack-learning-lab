# Aulas — FOUND-03.05 — Memória: stack e heap em nível introdutório

## Objetivo do tópico

Introduzir uma visão conceitual de memória para chamadas, valores e objetos.

## Conceitos do tópico

### FOUND-03.05.01 — Stack, frames e execução

#### Objetivo

Entender stack como organização conceitual das chamadas de função.

#### Pré-requisitos

Call stack.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Memória: stack e heap em nível introdutório**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Stack, frames e execução** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar que cada chamada possui um contexto temporário de execução.
