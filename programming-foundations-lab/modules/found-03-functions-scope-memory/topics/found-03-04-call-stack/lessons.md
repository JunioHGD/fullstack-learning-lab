# Aulas — FOUND-03.04 — Pilha de chamadas

## Objetivo do tópico

Entender como chamadas de função são organizadas durante a execução.

## Conceitos do tópico

### FOUND-03.04.01 — Call stack e ordem de execução

#### Objetivo

Compreender como chamadas são empilhadas e desempilhadas.

#### Pré-requisitos

Funções, retorno e escopo.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Pilha de chamadas**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Call stack e ordem de execução** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Rastrear a ordem de execução de funções chamadas em sequência ou aninhadas.

#### O que praticar

- Crie um exemplo simples em pseudocódigo ou código introdutório.
