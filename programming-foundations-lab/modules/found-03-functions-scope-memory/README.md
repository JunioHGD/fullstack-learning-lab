# FOUND-03 — Funções, Escopo e Memória

## Objetivo do módulo

Ensinar abstração por funções e introduzir escopo, chamada de função, passagem de dados e memória em nível conceitual.

## Pré-requisitos

Variáveis, tipos, operadores, expressões, condicionais e repetições.

## Tópicos do módulo

- `found-03-01-functions-parameters-return` — FOUND-03.01 — Definição de funções, parâmetros, argumentos e retorno
- `found-03-02-variable-scope` — FOUND-03.02 — Escopo de variáveis
- `found-03-03-parameter-passing-mutability` — FOUND-03.03 — Passagem de parâmetros, mutabilidade e efeitos colaterais
- `found-03-04-call-stack` — FOUND-03.04 — Pilha de chamadas
- `found-03-05-stack-and-heap-memory` — FOUND-03.05 — Memória: stack e heap em nível introdutório

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-03-functions-scope-memory/`.
