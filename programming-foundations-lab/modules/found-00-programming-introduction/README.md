# FOUND-00 — Introdução aos Fundamentos de Programação

## Objetivo do módulo

Apresentar a essência da programação, o papel das linguagens e o funcionamento básico de um programa antes de entrar em lógica, variáveis e código prático.

## Pré-requisitos

Nenhum conhecimento prévio de programação; noções básicas de raciocínio sequencial ajudam.

## Tópicos do módulo

- `found-00-01-programming-and-algorithms` — FOUND-00.01 — Programação e algoritmos
- `found-00-02-programming-languages` — FOUND-00.02 — Linguagens de programação e níveis de abstração
- `found-00-03-program-execution` — FOUND-00.03 — Execução de programas, compilação, interpretação e runtime
- `found-00-04-syntax-semantics-instructions` — FOUND-00.04 — Sintaxe, semântica e instruções
- `found-00-05-input-process-output` — FOUND-00.05 — Entrada, processamento e saída

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-00-programming-introduction/`.
