# Aulas — FOUND-00.05 — Entrada, processamento e saída

## Objetivo do tópico

Consolidar o modelo básico de funcionamento de programas a partir de entrada, processamento e saída.

## Conceitos do tópico

### FOUND-00.05.01 — Fluxo básico de um programa

#### Objetivo

Entender programas como transformação de entradas em saídas por meio de processamento.

#### Pré-requisitos

Noção de instruções e fluxo sequencial.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Entrada, processamento e saída**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Fluxo básico de um programa** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Identificar entrada, processamento e saída em problemas simples.
