# Aulas — FOUND-00.01 — Programação e algoritmos

## Objetivo do tópico

Construir a visão inicial de programação como resolução de problemas por meio de instruções organizadas.

## Conceitos do tópico

### FOUND-00.01.01 — O que é programação

#### Objetivo

Entender programação como a criação de instruções para resolver problemas com apoio do computador.

#### Pré-requisitos

Nenhum.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Programação e algoritmos**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **O que é programação** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar o que é programação e relacionar código, problema e solução.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Programação e algoritmos**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Algoritmos como sequência de passos** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado
