# Aulas — FOUND-00.02 — Linguagens de programação e níveis de abstração

## Objetivo do tópico

Entender o papel das linguagens e a diferença conceitual entre alto nível, baixo nível e código executável.

## Conceitos do tópico

### FOUND-00.02.01 — O que é uma linguagem de programação

#### Objetivo

Entender linguagens de programação como sistemas formais para expressar instruções computacionais.

#### Pré-requisitos

Noção de programa e algoritmo.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Linguagens de programação e níveis de abstração**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **O que é uma linguagem de programação** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar por que precisamos de linguagens para comunicar soluções ao computador.
