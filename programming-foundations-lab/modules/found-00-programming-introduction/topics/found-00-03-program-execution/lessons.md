# Aulas — FOUND-00.03 — Execução de programas, compilação, interpretação e runtime

## Objetivo do tópico

Compreender como código-fonte se transforma em execução e qual é o papel do ambiente de execução.

## Conceitos do tópico

### FOUND-00.03.01 — Compilação e interpretação

#### Objetivo

Entender a diferença geral entre compilar e interpretar código.

#### Pré-requisitos

Noção de linguagem e código-fonte.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Execução de programas, compilação, interpretação e runtime**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Compilação e interpretação** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar, em nível introdutório, como o código pode ser preparado ou lido para execução.
