# FOUND-07 — Paradigmas de Programação e Conceitos Avançados Introdutórios

## Objetivo do módulo

Apresentar estilos de programação e conceitos introdutórios que conectam os fundamentos a estudos futuros.

## Pré-requisitos

Funções, controle de fluxo, estruturas de dados básicas, depuração, testes e organização básica.

## Tópicos do módulo

- `found-07-01-imperative-declarative` — FOUND-07.01 — Paradigmas imperativo e declarativo
- `found-07-02-structured-modular-programming` — FOUND-07.02 — Programação estruturada e modular
- `found-07-03-object-oriented-intro` — FOUND-07.03 — Programação orientada a objetos introdutória
- `found-07-04-functional-programming-intro` — FOUND-07.04 — Programação funcional introdutória
- `found-07-05-complexity-future-structures` — FOUND-07.05 — Complexidade e estruturas de dados futuras

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-07-paradigms-intro-concepts/`.
