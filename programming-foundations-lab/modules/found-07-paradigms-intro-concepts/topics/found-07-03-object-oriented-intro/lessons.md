# Aulas — FOUND-07.03 — Programação orientada a objetos introdutória

## Objetivo do tópico

Introduzir objetos como agrupamentos de dados e comportamentos.

## Conceitos do tópico

### FOUND-07.03.01 — Objetos, classes e métodos

#### Objetivo

Compreender OOP como organização em torno de entidades com dados e comportamentos.

#### Pré-requisitos

Objetos, funções e estruturas de dados.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Programação orientada a objetos introdutória**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Objetos, classes e métodos** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar objeto, classe e método em nível introdutório.
