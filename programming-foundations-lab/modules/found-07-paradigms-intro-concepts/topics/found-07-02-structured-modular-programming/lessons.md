# Aulas — FOUND-07.02 — Programação estruturada e modular

## Objetivo do tópico

Conectar sequência, decisão, repetição e funções à organização estruturada do código.

## Conceitos do tópico

### FOUND-07.02.01 — Programação estruturada

#### Objetivo

Entender programação estruturada a partir de sequência, seleção e repetição.

#### Pré-requisitos

Controle de fluxo e funções.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Programação estruturada e modular**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Programação estruturada** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar como estruturas básicas substituem fluxos confusos.
