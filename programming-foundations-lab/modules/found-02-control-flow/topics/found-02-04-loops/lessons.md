# Aulas — FOUND-02.04 — Estruturas de repetição

## Objetivo do tópico

Repetir instruções com base em condições ou contadores.

## Conceitos do tópico

### FOUND-02.04.01 — while e do/while

#### Objetivo

Introduzir loops condicionais e condição de parada.

#### Pré-requisitos

Condicionais e expressões booleanas.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Estruturas de repetição**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **while e do/while** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Explicar como uma repetição continua ou termina conforme uma condição.
