# Aulas — FOUND-02.02 — Estruturas condicionais

## Objetivo do tópico

Executar blocos diferentes de código conforme condições avaliadas.

## Conceitos do tópico

### FOUND-02.02.01 — if e else

#### Objetivo

Introduzir decisões binárias com if e else.

#### Pré-requisitos

Expressões booleanas.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Estruturas condicionais**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **if e else** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Montar fluxos simples com dois caminhos possíveis.
