# FOUND-02 — Controle de Fluxo: Condições e Repetições

## Objetivo do módulo

Ensinar como controlar a ordem de execução por meio de decisões, repetições e comandos de fluxo.

## Pré-requisitos

Valores, tipos, variáveis, operadores, expressões e lógica booleana introdutória.

## Tópicos do módulo

- `found-02-01-boolean-logic` — FOUND-02.01 — Lógica booleana e tabelas verdade
- `found-02-02-conditionals` — FOUND-02.02 — Estruturas condicionais
- `found-02-03-switch-conditionals` — FOUND-02.03 — Condicionais múltiplas com switch
- `found-02-04-loops` — FOUND-02.04 — Estruturas de repetição
- `found-02-05-loop-control-validation` — FOUND-02.05 — Controle de loops, validações e invariantes

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-02-control-flow/`.
