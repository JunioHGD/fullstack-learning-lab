# Aulas — FOUND-06.03 — Testes manuais

## Objetivo do tópico

Verificar soluções com entradas, saídas esperadas e cenários representativos.

## Conceitos do tópico

### FOUND-06.03.01 — Casos de teste

#### Objetivo

Criar cenários de verificação com entrada e saída esperada.

#### Pré-requisitos

Funções, algoritmos e validações.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Testes manuais**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Casos de teste** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Definir casos normais, limites e inválidos para uma solução simples.
