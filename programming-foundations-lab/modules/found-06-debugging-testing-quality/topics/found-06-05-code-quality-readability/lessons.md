# Aulas — FOUND-06.05 — Qualidade, legibilidade e organização básica

## Objetivo do tópico

Desenvolver hábitos iniciais de código claro, organizado e fácil de manter.

## Conceitos do tópico

### FOUND-06.05.01 — Nomes claros e legibilidade

#### Objetivo

Usar nomes de variáveis e funções que comuniquem intenção.

#### Pré-requisitos

Variáveis, funções e estruturas básicas.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Qualidade, legibilidade e organização básica**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Nomes claros e legibilidade** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Melhorar a leitura de trechos simples por meio de nomes adequados.
