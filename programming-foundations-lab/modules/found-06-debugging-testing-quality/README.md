# FOUND-06 — Depuração, Testes e Qualidade de Código

## Objetivo do módulo

Ensinar práticas iniciais para encontrar erros, testar comportamentos e melhorar legibilidade e organização.

## Pré-requisitos

Programação básica consolidada com funções, controle de fluxo, estruturas de dados e algoritmos simples.

## Tópicos do módulo

- `found-06-01-debugging-errors` — FOUND-06.01 — Identificação de erros e depuração
- `found-06-02-error-handling` — FOUND-06.02 — Tratamento de erros introdutório
- `found-06-03-manual-tests` — FOUND-06.03 — Testes manuais
- `found-06-04-basic-automated-tests` — FOUND-06.04 — Testes automatizados básicos
- `found-06-05-code-quality-readability` — FOUND-06.05 — Qualidade, legibilidade e organização básica

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-06-debugging-testing-quality/`.
