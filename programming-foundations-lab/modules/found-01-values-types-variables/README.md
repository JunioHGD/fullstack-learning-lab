# FOUND-01 — Valores, Tipos de Dados e Variáveis

## Objetivo do módulo

Ensinar os blocos básicos usados para representar, armazenar e manipular dados em programas.

## Pré-requisitos

Conceitos de programa, algoritmo, linguagem, sintaxe, semântica e fluxo entrada-processamento-saída.

## Tópicos do módulo

- `found-01-01-values-and-data-types` — FOUND-01.01 — Valores e tipos de dados
- `found-01-02-variables-constants-identifiers` — FOUND-01.02 — Variáveis, constantes e identificadores
- `found-01-03-operators-and-expressions` — FOUND-01.03 — Operadores e expressões
- `found-01-04-expressions-statements-evaluation` — FOUND-01.04 — Expressões, instruções e avaliação
- `found-01-05-mutability-and-data-passing` — FOUND-01.05 — Mutabilidade e passagem de dados

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-01-values-types-variables/`.
