# Aulas — FOUND-01.01 — Valores e tipos de dados

## Objetivo do tópico

Compreender que programas manipulam valores e que esses valores possuem categorias de uso.

## Conceitos do tópico

### FOUND-01.01.01 — Valores em programação

#### Objetivo

Entender valor como dado manipulado pelo programa durante a execução.

#### Pré-requisitos

Modelo entrada-processamento-saída.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Valores e tipos de dados**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Valores em programação** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Reconhecer valores em exemplos simples de código e pseudocódigo.
