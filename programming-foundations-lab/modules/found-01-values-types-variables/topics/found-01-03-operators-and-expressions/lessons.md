# Aulas — FOUND-01.03 — Operadores e expressões

## Objetivo do tópico

Entender como operadores combinam valores e variáveis para produzir resultados.

## Conceitos do tópico

### FOUND-01.03.01 — Operadores aritméticos

#### Objetivo

Compreender operações matemáticas básicas aplicadas a valores numéricos.

#### Pré-requisitos

Valores, tipos e variáveis.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Operadores e expressões**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Operadores aritméticos** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Prever resultados de expressões aritméticas simples.
