# Aulas — FOUND-05.01 — Pensamento computacional

## Objetivo do tópico

Usar pensamento computacional para transformar problemas em soluções programáveis.

## Conceitos do tópico

### FOUND-05.01.01 — Problemas, entradas, regras e saídas

#### Objetivo

Identificar os componentes essenciais de um problema antes de resolver.

#### Pré-requisitos

Entrada-processamento-saída e estruturas básicas.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Pensamento computacional**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Problemas, entradas, regras e saídas** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Separar dados de entrada, regras de processamento e saída esperada.
