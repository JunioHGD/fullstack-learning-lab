# Aulas — FOUND-05.02 — Decomposição, padrões e abstração

## Objetivo do tópico

Dividir problemas, identificar repetições e focar nos elementos relevantes da solução.

## Conceitos do tópico

### FOUND-05.02.01 — Decomposição de problemas

#### Objetivo

Quebrar problemas em subproblemas menores e tratáveis.

#### Pré-requisitos

Pilares do pensamento computacional.

#### Explicação inicial

Este conceito deve ser estudado como parte do tópico **Decomposição, padrões e abstração**. O foco é compreender a ideia central, identificar quando ela aparece em problemas simples e conseguir explicá-la com exemplos próprios.

#### O que praticar

- Explique o conceito de **Decomposição de problemas** com suas próprias palavras.
- Crie um exemplo simples em pseudocódigo ou código introdutório.
- Identifique um erro comum relacionado ao conceito.
- Relacione este conceito com os tópicos anteriores do módulo.

#### Resultado esperado

Criar uma lista de etapas menores para resolver um problema.
