# FOUND-05 — Pensamento Computacional e Resolução de Problemas

## Objetivo do módulo

Desenvolver decomposição, reconhecimento de padrões, abstração, modelagem e raciocínio algorítmico básico.

## Pré-requisitos

Controle de fluxo, funções, arrays, strings, objetos e validações simples.

## Tópicos do módulo

- `found-05-01-computational-thinking` — FOUND-05.01 — Pensamento computacional
- `found-05-02-decomposition-patterns-abstraction` — FOUND-05.02 — Decomposição, padrões e abstração
- `found-05-03-pseudocode-flowcharts` — FOUND-05.03 — Pseudocódigo e fluxogramas
- `found-05-04-basic-algorithms` — FOUND-05.04 — Algoritmos básicos de busca e ordenação conceitual
- `found-05-05-basic-complexity-validation` — FOUND-05.05 — Complexidade básica e validação de algoritmos

## Como usar este módulo

- Use `checklist.md` para acompanhar o progresso do módulo.
- Use `interview-questions.md` para revisar o módulo de forma integrada.
- Acesse cada pasta em `topics/` para estudar os conceitos, responder perguntas e criar exercícios.
- Ao concluir o módulo, desenvolva o projeto correspondente em `docs/projects/found-05-computational-thinking/`.
