# Logic & JavaScript Lab

Laboratório de estudos que integra lógica de programação, pensamento computacional e JavaScript em uma única progressão curricular.

## Objetivos

- desenvolver raciocínio lógico e capacidade de resolução de problemas;
- compreender fundamentos universais de programação;
- aplicar os conceitos diretamente em JavaScript;
- evoluir de forma progressiva em organização, testes, depuração e qualidade de código.

## Organização

O conteúdo está distribuído em blocos e unidades. Cada unidade contém uma referência curta, perguntas de assimilação e exercícios. Ao final de cada bloco, há uma entrevista técnica e um projeto integrador.

## Fluxo de estudo

1. Estudar a unidade.
2. Consultar as fontes recomendadas.
3. Responder às perguntas de assimilação.
4. Resolver os exercícios.
5. Realizar a entrevista técnica do bloco.
6. Desenvolver o projeto integrador.

O nome do repositório está em inglês. A documentação e a organização interna estão em português, preservando termos oficiais do JavaScript.
