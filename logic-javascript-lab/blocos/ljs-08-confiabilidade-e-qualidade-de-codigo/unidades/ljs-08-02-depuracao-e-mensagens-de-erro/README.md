# LJS-08.02 — Depuração e mensagens de erro

- **Foco:** Lógica
- **Prioridade:** Essencial
- **Situação:** Pendente

## Objetivo

Investigar falhas por meio de mensagens, rastreamento e inspeção de valores.

## Conceitos

- stack trace
- console.log
- console.error
- console.table
- breakpoints
- debugger

## Resultado esperado

Localizar a origem de erros e acompanhar o estado do programa durante a execução.
